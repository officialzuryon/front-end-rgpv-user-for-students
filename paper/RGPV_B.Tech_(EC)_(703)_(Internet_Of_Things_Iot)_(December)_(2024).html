Internet Of Things Iot

Course: B.Tech Code: 703 Branch: EC Date: December 2024

EC-703 (B) (GS)

B.Tech., VII Semester

Examination, December 2024

Grading System (GS)

Internet of Things (IoT)

Time: Three Hours Maximum Marks: 70

Note: i) Attempt any five questions.

ii) All questions carry equal marks.

iii) In case any doubt or dispute the English version question should be treated as final.

iv) किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.

a) Explain IoT Architectural view.

IoT वास्तुशिल्प दृश्य को समझाइये।

b) What are the digital and physical value-creation layers in an Internet of Things application? Explain it.

इंटरनेट ऑफ़ थिंग्स एप्लीकेशन में डिजिटल और फिजिकल मूल्य-निर्माण परतें क्या है? इसे स्पष्ट करे।

2.

Describe how a solution is designed for a particular problem by making use applied architecture in M2M/IoT. Also, explain the use of partitioning the architectural work and solution work into two domains.

3.

a) Explain with example MQTT Protocol. What is role of MQTT protocol in IoT?

MQTT प्रोटोकॉल को उदाहरण सहित समझाइये। IoT में MQTT प्रोटोकॉल की क्या भूमिका है?

b) Explain the Constrained Application Protocol (CoAP).

कंस्ट्रैंड एप्लीकेशन प्रोटोकॉल (CoAP) को समझाइए।

4.

a) What is wireless sensor network? Discuss the benefits and limitations of wireless sensor networks.

वायरलेस सेंसर नेटवर्क क्या है? वायरलेस सेंसर नेटवर्क के लाभों और सीमाओं पर चर्चा करें।

b) Explain Actuator and its use in IoT? Explain types of actuators in detail.

एक्चुएटर और IoT में इसके उपयोग के बारे में बताइए। एक्चुएटर्स के प्रकारों को विस्तार से समझाइये।

5.

a) Write note on programming with Arduino IDE. Write Arduino code that uses ultrasonic sensor to measure distance of an object from the sensor.

Arduino IDE के साथ प्रोग्रामिंग पर नोट लिखे। Arduino कोड लिखे जो सेंसर से किसी वस्तु की दूरी नापने के लिए अल्ट्रासोनिक सेंसर का उपयोग करता है।

b) Discuss the design objectives of IoT architecture needed to target a human system of real-world services.

वास्तविक दुनिया की सेवाओं की हीमैन प्रणाली को लक्षित करने के लिए आवश्यक IoT आर्किटेक्चर के डिजाइन उद्देश्यों पर चर्चा करें।

EC-703 (B) (GS) PTO
[3]
6.

a) Explain how cloud of things acts as an enabler for new value added services and applications with a neat diagram.

स्पष्ट चित्र के साथ बताइये कि क्लाउड ऑफ थिंग्स नई मूल्य वर्धित सेवाओं और अनुप्रयोगों के लिए एक प्रवर्तक के रूप में कैसे कार्य करता है।

b) Explain the deployment and operational view, resources, services, virtual entities, users in an IoT system by considering a Parking IoT example.

पार्किंग IoT उदाहरण पर विचार करके IoT प्रणाली में तैनाती और परिचालन दृश्य, संसाधनों, सेवाओं, आभासी संस्थाओं, उपयोगकर्ताओं की व्याख्या करें।

7.

a) Differentiate between Industrial IoT and Automotive IoT.

औद्योगिक IoT और ऑटोमोटिव IoT के बीच अंतर बताइए।

b) With neat diagrams describe about how Network Function Virtualization virtualizes the IoT devices?

स्वच्छ आरेखों के साथ वर्णन करें कि नेटवर्क फंक्शन वर्चुअलाइजेशन IoT उपकरणों को कैसे वर्चुअलाइज करता है?

8.

Write a short note on any two:

  • i) IoT conceptual framework
  • ii) SDN
  • iii) SOAP
  • iv) IP addressing in IoT

किन्हीं दो पर संक्षिप्त टिप्पणी लिखें।

  • i) IoT वैचारिक ढाँचा
  • ii) SDN
  • iii) SOAP
  • iv) IoT में IP एड्रेसिंग
