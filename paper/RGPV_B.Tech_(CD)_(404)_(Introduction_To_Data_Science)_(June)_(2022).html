1. a)

Briefly explain the role of Data Science in various fields.

विभिन्न क्षेत्रों में डाटा साइंस की भूमिका को संक्षेप में समझाइए।

डाटा साइंस में शामिल डाटा संग्रह के विभिन्न तरीकों के बारे में लिखें।

3. a)

Explain the need for Correlation and Regression analysis in Data analysis with an example.

डाटा विश्लेषण में सहसंबध और प्रतिगमन विश्लेषण की आवश्यकता को एक उदाहरण के साथ समझाइए।

b)

How outliers are identified and handled in variables in a Data Set - Explain.

How is model evaluation done by using Data Visualization? Give an example.

डाटा विजुअलाइजेशन का उपयोग करके मॉडल मूल्यांकन कैसे किया जाता है? एक उदाहरण दें।

b)

Explain briefly the various models involved in Regression Analysis.

रिग्रेशन एनालिसिस में शामिल विभिन्न मॉडलों को संक्षेप में समझाइए।

5. a)

Explain how overfitting and underfitting issues are handled in Regression modeling.

बताएं कि रिग्रेशन मॉडलिंग में ओवरफिटिंग और अंडरफिटिंग मुद्दों को कैसे संभाला जाता है?

b)

Explain the procedure of doing prediction analysis using Ridge Regression Model.

रिज रिग्रेशन मॉडल का उपयोग करते हुए भविष्यवाणी विश्लेषण करने की प्रक्रिया की व्याख्या करें।

6. a)

What is meant by the Pivot Table? How it is useful in exploring data analytics?

पिवट टेबल से क्या तात्पर्य है? डाटा एनालिटिक्स की खोज में यह कैसे उपयोगी है?

b)

What is ANOVA? Why it is so important in Analysis of variance and how it is used in statistics?

एनोवा क्या है? विचरण के विश्लेषण में यह इतना महत्वपूर्ण क्यों है और इसका उपयोग आंकड़ों में कैसे किया जाता है?

7. a)

Explain Data Security issues with an example.

डाटा सुरक्षा मुद्दों की एक उदाहरण के साथ व्याख्या करें।

b)

Explain the use of Grid Search in testing Multiple parameters.

एकाधिक मापदंडों के परीक्षण में ग्रिड खोज के उपयोग की व्याख्या करें।
