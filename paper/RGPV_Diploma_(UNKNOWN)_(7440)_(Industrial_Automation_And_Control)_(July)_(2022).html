1.

Choose the correct answer.

सही उत्तर का चयन कीजिए।

(a)

What is the difference between a PLC and a DCS?

(a) PLCs are used to control machines while DCSs are used to control entire plants.

(b) PLCs are used to control machines while DCSs are used to control networks.

(c) PLCs are used to control machines while DCSs are used to control buildings.

(d) PLCs are used to control machines while DCSs are used to control vehicle.

2 each
(i)

PLC और DCS के बीच क्या अंतर है?

(अ) PLC का उपयोग मशीनों को नियंत्रित करने के लिए किया जाता है जबकि DCS का उपयोग पूरे संयंत्र को नियंत्रित करने के लिए किया जाता है।

(ब) PLC का उपयोग मशीनों को नियंत्रित करने के लिए किया जाता है जबकि DCS का उपयोग नेटवर्क को नियंत्रित करने के लिए किया जाता है।

(स) PLC का उपयोग मशीनों को नियंत्रित करने के लिए किया जाता है जबकि DCS का उपयोग इमारतों को नियंत्रित करने के लिए किया जाता है।

(द) PLC का उपयोग मशीनों को नियंत्रित करने के लिए किया जाता है जबकि DCS का उपयोग वाहनों को नियंत्रित करने के लिए किया जाता है।

(ii)

What type of programming language is typically used to program a PLC?

(a) C++

(b) Java

(c) Ladder Logic

(d) Python

PLC को प्रोग्राम करने के लिए आमतौर पर किस प्रकार की प्रोग्रामिंग भाषा का उपयोग किया जाता है?

(अ) C++

(स) लेडर लॉजिक

(ब) जावा

(द) पायथन

(iii)

Which of the following are the criteria for selecting a programmable logic controller.

i) Input and output capacity

ii) Size of memory needed

iii) Speed and power of CPU

iv) Physical and environmental conditions

(a) (i) and (ii)

(b) (i) and (iii)

(c) (i), (ii) and (iii)

(d) (i), (ii), (iii) and (iv)

प्रोग्रामेबल लॉजिक कंट्रोलर को चुनने के लिए निम्नलिखित में से कौन-सा मानदंड सही है?

i) इनपुट और आउटपुट क्षमता

ii) आवश्यक मेमोरी का आकार

iii) CPU की गति और शक्ति

iv) भौतिक और पर्यावरणीय स्थितियाँ

(अ) (i) और (ii)

(स) (i), (ii) और (iii)

(ब) (i) और (iii)

(द) (i), (ii), (iii) और (iv)

(iv)

Which of the following is NOT a component of the SCADA system?

(a) Master terminal unit

(b) Graphical terminal unit

(c) Remote terminal unit

(d) Communication network

निम्नलिखित में से कौन SCADA प्रणाली का घटक नहीं है?

(अ) मास्टर टर्मिनल यूनिट

(ब) ग्राफिकल टर्मिनल यूनिट

(स) रिमोट टर्मिनल यूनिट

(द) संचार नेटवर्क

(v)

V/f is maintained constant in the following case of speed control of induction motor.

(a) Below base speed with voltage control

(b) Below the base speed with frequency control

(c) Above base speed with frequency control

(d) None of these

प्रेरण मोटर की गति नियंत्रण के निम्नलिखित मामले में V/f को स्थिर बनाए रखा जाता है

(अ) वोल्टेज नियंत्रण के साथ आधार गति से नीचे

(ब) आवृत्ति नियंत्रण के साथ आधार गति से नीचे
