M.C.A. II Semester (Two Year Course)
Examination, June 2024
Computer Network
Time: Three Hours
Maximum Marks: 70
Note: i) Attempt any five questions.
किन्हीं पाँच प्रश्नों को हल कीजिए।
ii) All questions carry equal marks.
सभी प्रश्नों के समान अंक हैं।
iii) In case of any doubt or dispute the English version question should be treated as final.
किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।
1. a)
Define the term Computer Network. Enlist and explain goals of computer network.
कम्प्यूटर नेटवर्क शब्द को परिभाषित करें। कम्प्यूटर नेटवर्क के लक्ष्यों को सूचीबद्ध करें और समझाइए।
7
b)
Explain OSI layered architecture. Compare OSI and TCP/IP reference model.
OSI स्तरित आर्किटेक्चर की व्याख्या करें। OSI और TCP/IP संदर्भ मॉडल की तुलना करें।
7
2. a)
Write short note on following (any two)
  1. DNS
  2. SNMP
  3. Virtual terminal protocol
7
3. a)
Define Hamming code. Find Hamming code for 100010 message string.
हैम्मिंग कोड परिभाषित करें। 100010 संदेश स्ट्रिंग के लिए हैम्मिंग कोड खोजें।
7
b)
Consider an error-free 64 kbps satellite channel used to send 512 byte data frames in one direction with very short acknowledgements coming back the other way. What is maximum throughput for window size 1, 7, 15 and 127 for sliding window protocol?
एक त्रुटि-रहित 64 एमबीपीएस सैटेलाइट चैनल पर विचार करें जिसका उपयोग एक दिशा में 512 बाइट डेटा फ्रेम भेजने के लिए किया जाता है, जिसमें दूसरी दिशा में बहुत कम पावती आती है। स्लाइडिंग विंडो प्रोटोकॉल के लिए विंडो आकार 1, 7, 15 और 127 के लिए अधिकतम थ्रूपुट क्या है?
7
4. a)
Explain working of token ring protocol.
टोकन रिंग प्रोटोकॉल की कार्यप्रणाली समझाइए।

5. a)
Explain difference between Repeater, Bridge, switch and Routers.
रिपीटर, ब्रिज, स्विच और राउटर के बीच अंतर समझाइए।
7
b)
Explain working of Go-Back-N sliding window protocol.
गो-बैक-एन स्लाइडिंग विंडो प्रोटोकॉल की कार्यप्रणाली समझाइए।
7
c)
Define routing. Enlist types of routing algorithm. Compare adaptive routing approach over non adaptive routing.
राउटिंग को परिभाषित करें। राउटिंग एल्गोरिदम के प्रकारों को सूचीबद्ध करें। अनुकूली राउटिंग दृष्टिकोण और गैर अनुकूली राउटिंग के बीच तुलना करें।
7
6. a)
Explain working of open shortest path first routing algorithm.
ओपन शॉर्टेस्ट पाथ फर्स्ट राउटिंग एल्गोरिथम की कार्यप्रणाली समझाइए।
7
b)
Define congestion in network. Explain any one congestion control method in network.
नेटवर्क में भीड़भाड़ को परिभाषित करें। नेटवर्क में भीड़भाड़ नियंत्रण की किसी एक विधि की व्याख्या करें।
7
7. a)
Explain in detail connection oriented transport layer protocol. Differentiate between TCP and UDP.
कनेक्शन ओरिएंटेड ट्रांसपोर्ट लेयर प्रोटोकॉल को विस्तार से समझाइए। TCP और UDP के बीच अंतर बताइए।
7
b)
Explain the architecture and reference model of HIPERLAN-2 in detail.
HIPERLAN-2 की वास्तुकला और संदर्भ मॉडल को विस्तार से समझाइए।
7
