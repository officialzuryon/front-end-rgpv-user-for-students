Pharmacognosy

F/2023/8103
Total Pages : 2
[2]

Time : Three Hours

Maximum Marks : 80

First Year Pharmacy

PHARMACOGNOSY

(ER20-13T)

Note :

  1. Attempt any five questions.
  2. All questions carry equal marks.
1.

Attempt any two.

a)
What are fibers? Write the classification of fibers.
b)
Define the following terms:
  1. Vein islet number
  2. Refractive index
  3. Stomatal index
  4. Optical rotation
c)
Define the tannin and give the classification and identification test for tannin.
2.

Attempt any two.

a)
Write in detail about the phytochemical test used for glycosides and carbohydrates.
b)
Give a detailed account on herbal formulation.
c)
Write the biological source, chemical constituents and therapeutic uses of
  1. Digitalis
  2. Senna
3

Write short note on : (Any four)

a)
Therapeutic application of Nutraceuticals.
b)
Resins and resin combination.
c)
Sandal wood oil
d)
Scope of pharmacognosy
e)
Macroscopic and microscopic character of fennel.
4.

Attempt any two.

a)
Define evaluation of crude drugs. Discuss in detail physical evaluation.
b)
Define the following terms:
  1. Admixture
  2. Sophistication
  3. Deterioration
  4. Spoilage
c)
Write uses of Churna, Bhasma and Lehya.
5.

Attempt any two.

a)
Differentiate between the following:
  1. Volatile and fixed oils
  2. Organized and unorganized drugs
b)
Name methods for classification of natural drugs. Discuss in detail pharmacological classification with an example.
c)
Give a brief account on phytochemical investigation of drugs.
6.

Write short note on any four with an example.

a)
Pharmaceutical aids
b)
Anti-hypertensives
c)
Cardio tonic
d)
Vitamins
e)
Anti-dysenteric
7.

Attempt any two.

a)
Write a brief note on methods of adulteration of crude drugs.
b)
What are sutures? Write a note on surgical catgut.
c)
Write the therapeutic applications of probiotics and prebiotics.
8.

Attempt any two.

a)
Write therapeutic and cosmetic uses of Olive oil, Aloe Vera gel.
b)
Write a note on Ashwagandha and Tulsi.
c)
What are the basic principles involved in the traditional systems of medicine like Homeopathy.
F/2023/8103
