MCA-204

M.C.A. II Semester (Two Year Course)

Examination, June 2024

Algorithm Design

Time : Three Hours

Maximum Marks : 70

Note:

i) Attempt any five questions.
किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.
सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.
किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.

a) What is a circular linked list? What are the different operations performed on it, explain with an example.

सर्कुलर लिंक्ड लिस्ट क्या है? इस पर कौन-कौन से अलग-अलग आपरेशन किए जाते है, उदाहरण देकर समझाइए।

b) Write and discuss various applications of stack and queue data structure in computer science.

कंप्यूटर विज्ञान में स्टैक और क्यू डेटा संरचना के विभिन्न अनुप्रयोगों को लिखें और चर्चा करें।

2.

a) Convert A+(B*C-(D/E)*F) infix expression into postfix format showing stack status every after stack in table.

तालिका में प्रत्येक स्टैक के बाद स्टैक स्थिति दिखाते हुए A+(B*C-(D/E)*F) इन्फिक्स अभिव्यक्ति को पोस्टफिक्स प्रारूप में परिवर्तित करें।

b) Write down the logic to implement two stack using one array.

एक ऐरे का उपयोग करके दो स्टैक को कार्यान्वित करने के लिए तर्क लिखें।

3.

a) Draw binary tree using following preorder and inorder traversing.

निम्नलिखित प्रीऑर्डर और इनऑर्डर ट्रैवर्सिंग का उपयोग करके बाइनरी ट्री बनाएँ।

Preorder - GBQACKFPDERH

Inorder - QBKCFAGPEDHR

प्रीऑर्डर - GBQACKFPDERH

इनऑर्डर - QBKCFAGPEDHR

b) What are the application of tree data structure? Explain.

ट्री डेटा संरचना के अनुप्रयोग क्या हैं? समझाइए।

4.

a) What is hashing? What are the qualities of a good hash function? Explain any two has function in detail.

हैशिंग क्या है? एक अच्छे हैश फ़ंक्शन के गुण क्या हैं? किसी भी दो हैश फ़ंक्शन को विस्तार से समझाइए।

b) Sort the following array using quick-sort Algorithm.

(5, 8, 3, 9, 2, 10, 1, 45, 32)

क्विक-सॉर्ट एल्गोरिदम का उपयोग करके निम्नलिखित सरणी को क्रमबद्ध करें।

(5, 8, 3, 9, 2, 10, 1, 45, 32)

5.

a) What is minimum spanning Tree? Find the minimum spanning tree of the following graph.

न्यूनतम स्पैनिंग ट्री क्या है? निम्नलिखित ग्राफ में न्यूनतम स्पैनिंग ट्री ज्ञात कीजिए।

b) Differentiate between Depth-first and breadth-first traversal Algorithm.

गहराई-प्रथम और चौड़ाई-प्रथम ट्रैवर्सल एल्गोरिदम के बीच अंतर बताइए।

6.

a) Illustrate merge sort algorithm and discuss time complexity for the following data.

78, 32, 42, 62, 98, 12, 34, 83

मर्ज सॉर्ट एल्गोरिदम का चित्रण करें और निम्नलिखित डेटा के लिए समय जटिलता पर चर्चा करें।

78, 32, 42, 62, 98, 12, 34, 83

b) Compute the optimal solution for knapsack problem using Greedy method n = 4, m = 15, p = (10, 10, 12, 18), w = (2, 4, 6, 9)

ग्रीडी विधि का उपयोग करके नैपसैक समस्या के लिए इष्टतम समाधान की गणना करें n = 4, m = 15, p = (10, 10, 12, 18), w = (2, 4, 6, 9)

7.

a) Solve the 4-queen problem using back tracking.

बैक ट्रैकिंग का उपयोग करके 4-क्वीन समस्या को हल करें।

b) What do you understand by NP-hard and NP-complete problem?

NP- हार्ड और NP- पूर्ण समस्या से आप क्या समझते हैं?

Write short notes on any three of the following.

a) Traveling Salesman Problem

अ) ट्रैवलिंग सेल्समैन समस्या

b) B+ Tree

ब) B+ ट्री

c) Topological Sort

स) टोपोलॉजिकल सॉर्ट

d) Dynamic Programming
