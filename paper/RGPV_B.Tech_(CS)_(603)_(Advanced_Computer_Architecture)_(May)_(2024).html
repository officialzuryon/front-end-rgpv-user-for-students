किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1. a)
Describe the four categories in Flynn's classification of computer architecture. Give a brief description of every category? फ़्लिन के कंप्यूटर आर्किटेक्चर के वर्गीकरण में चार श्रेणियों का वर्णन करें। प्रत्येक श्रेणी का संक्षिप्त विवरण दें?
7
b)
Differentiate between fine-grained and coarse-grained parallelism. Discuss the types of latency present in computer architecture? बारीक दाने वाली और मोटे दाने वाली समानता के बीच अंतर करें। कंप्यूटर आर्किटेक्चर में मौजूद विलंबता के प्रकारों पर चर्चा करें।
7
2. a)
Compare and contrast different static interconnection network topologies in computer architecture such as mesh, tree and hypercube with neat diagrams? कंप्यूटर आर्किटेक��चर में अलग-अलग स्टैटिक इंटरकनेक्शन नेटवर्क टोपोलॉजी जैसे मेश, ट्री और हाइपरक्यूब की साफ-सुथरे आरेखों के साथ तुलना करें?
7
b)
Define memory hierarchy in computer architecture. List the different levels of memory hierarchy and describe their characteristics? कंप्यूटर आर्किटेक्चर में मेमोरी पदानुक्रम को परिभाषित करें। स्मृति पदानुक्रम के विभिन्न स्तरों की सूची बनाएं और उनकी विशेषताओं का वर्णन करें?
7
3. a)
Define interleaved memory organization and memory interleaving. Describe how memory interleaving improves memory access performance in computer systems? इंटरलीव्ड मेमोरी संगठन और मेमोरी इंटरलीविंग को परिभाषित करें। वर्णन करें कि मेमोरी इंटरलीविंग कंप्यूटर सिस्टम में मेमोरी एक्सेस प्रदर्शन को कैसे बेहतर बनाती है?
7
b)
Describe the basic architecture of a backplane bus system. How does it facilitate communication between different modules and components within a computer system? बैकप्लेन बस प्रणाली की बुनियादी वास्तुकला का वर्णन करें। यह कंप्यूटर सिस्टम के भीतर विभिन्न मॉड्यूल और घटकों के बीच संचार को कैसे सुविधाजनक बनाता है?
7
4. a)
Compare and contrast a nonlinear pipeline processor with a linear pipeline processor in terms of their ability to exploit instruction-level parallelism? निर्देश-स्तरीय समानता का फायदा उठाने की उनकी क्षमता के संदर्भ में एक नॉनलिनियर पाइपलाइन प्रोसेसर की एक रैखिक पाइपलाइन प्रोसेसर के साथ तुलना करें।
7
b)
What is Dynamic Instruction Scheduling, particularly in the context of "score boarding"? विशेष रूप से "स्कोर बोर्डिंग" के संदर्भ में डायनामिक इंस्ट्रक्शन शेड्यूलिंग क्या है?
7
5. a)
Design a simple arithmetic pipeline with different stages. Provide a brief overview of each stage's function? विभिन्न चरणों वाली एक सरल अंकगणितीय पाइपलाइन डिज़ाइन करें। प्रत्येक चरण के कार्य का संक्षिप्त विवरण प्रदान करें।
7
b)
List and briefly describe the main types of message routing schemes used in multicomputer networks. मल्टीकंप्यूटर नेटवर्क में उपयोग की जाने वाली मुख्य प्रकार की संदेश रूटिंग योजनाओं की सूची बनाएं और उनका संक्षेप में वर्णन करें।
7
6. a)
Define vector supercomputer architecture. What distinguishes it from other types of supercomputer architectures? वेक्टर सुपरकंप्यूटर आर्किटेक्चर को परिभाषित करें। इसे अन्य प्रकार के सुपरकंप्यूटर आर्किटेक्चर से क्या अलग करता है?
7
b)
Explain how data is shared among processing elements in a shared memory SIMD system? What mechanisms are used to ensure data consistency? बताएं कि साझा मेमोरी SIMD सिस्टम में प्रोसेसिंग तत्वों के बीच डेटा कैसे साझा किया जाता है। डेटा स्थिरता सुनिश्चित करने के लिए किन तंत्रों का उपयोग किया जाता है?
7
7. a)
What is the shared-variable model in parallel programming? Describe its basic characteristics? समानांतर प्रोग्रामिंग में साझा-चर मॉडल क्या है? इसकी मूलभूत विशेषताओं का वर्णन करें।
7
b)
List and briefly explain fundamental concepts related to parallel programming environments, such as parallel compilers and debugging tools? समानांतर प्रोग्रा��िंग वातावरण, जैसे समानांतर कंपाइलर्स और डिबगिंग टूल से संबंधित मूलभूत अवधारणाओं की सूची बनाएं और उन्हें संक्षेप में समझाएं।
7
Write short notes on any two: किन्हीं दो पर संक्षिप्त टिप्पणी लिखिए।
