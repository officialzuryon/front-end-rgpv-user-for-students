1.
a)
Find a real root for ex sin x = 1 using Regular False method. 7

नियमित गलत विधि का उपयोग करके ex sin x = 1 के लिए वास्तविक मूल खोजें।

b)
Find a real root of the equation x - cos x = 0 which lies between 0 and 1 using Bisection method. 7

द्विभाजन विधि का उपयोग करके समीकरण x - cos x = 0 का वास्तविक मूल ज्ञात करें जो 0 और 1 के बीच स्थित है।

a)
From the following table estimate the number of person's earning wages between 60-70 rupees by using Newton's forward interpolation formula. 7

निम्नलिखित तालिका से न्यूटन के फॉरवर्ड इंटरपोलेशन फॉर्मूले का उपयोग करके 60-70 रुपये के बीच व्यक्ति की कमाई मजदूरी की संख्या का अनुमान लगाएं।

b)
Evaluate ∫01 sin x dx by dividing the range into 10 equal parts using Simpson's 1/3 rule. 7

सिम्पसन के 1/3 नियम का उपयोग करके रेंज को 10 बराबर भागों में विभाजित करके ∫01 sin x dx का मूल्यांकन करें।

3.
a)
Find the solution of the system of equations by using Gauss - Jacobi's method correct to 4 decimal places. 14

गॉस-जेकोबी विधि का उपयोग करके दशमलव के 4 स्थानों तक सही समीकरणों की प्रणाली का समाधान खोजें।

10x - 2y - z = 3, -2x + 10y - z = 15, -x - y + 10z = 27
4.
a)
Find y(0.1), y(0.2), y(0.3) using Taylor's series method given that dy/dx = y2 + x and y(0) = 1. 14

टेलर की श्रृंखला विधि का उपयोग करके y(0.1), y(0.2), y(0.3) खोजें,

बशर्ते कि dy/dx = y2 + x और y(0) = 1 हो।

5.
a)
Explain One dimensional Wave and Heat equation's. 7

एक आयामी तरंग और ऊष्मा समीकरण की व्याख्या करें।

b)
Using Laplace transform, Evaluate ∫0∞ e-2t sin t dt. 7

लाप्लास ट्रांसफॉर्म का उपयोग करके, ∫0∞ e-2t sin t dt का मूल्यांकन करें।

6.
a)
Using Convolution theorem, Find L-1 [ 1 / (s2 + a2)2 ]. 7

कनवोल्यूशन प्रमेय का उपयोग करके L-1 [ 1 / (s2 + a2)2 ] खोजें।

b)
Using Laplace transform, Solve (D2 + 4D + 5) y = 5, given that y(0) = 0, y'(0) = 0. 7

लाप्लास ट्रांसफॉर्म का उपयोग करके, (D2 + 4D + 5) y = 5 को हल करें, यह देखते हुए कि y(0) = 0, y'(0) = 0 हो।

7.
a)
Derive mean of the Normal Distribution. 7

सामान्य वितरण का माध्य प्राप्त करें।

b)
2% of the items of a factory are defective. The items are packed in boxes. What is the probability that there will be at least three defective items in a box of 100 items? 7

एक कारखाने की 2% वस्तुएँ खराब हैं। सामान बक्सों में पैक किया गया है। इसकी क्या प्रायिकता है कि 100 वस्तुओं के एक डिब्बे में कम से कम तीन खराब वस्तुएँ होंगी?
