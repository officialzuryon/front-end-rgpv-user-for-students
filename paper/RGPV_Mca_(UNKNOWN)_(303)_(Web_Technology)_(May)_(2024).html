MCA-303 (2)

M.C.A. III Semester (Two Year Course)

Examination, May 2024

Web Technology

(Elective-I)

Time : Three Hours Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1. a)

Explain in detail about the role of ISP and DNS in Internet with the help of an example.

एक उदाहरण की सहायता से इंटरनेट में ISP और DNS की भूमिका के बारे में बताइए।

b)

What do you mean by planning and publishing websites? Explain.

वेबसाइटों की योजना और प्रकाशन से आपका क्या तात्पर्य है? व्याख्या करें।

2. a)

Write in detail about need, basic syntax and structure of Cascading Style Sheets (CSS).

कैस्केडिंग स्टाइल शीट्स (CSS) की आवश्यकता, मूल वाक्यविन्यास और संरचना के बारे में विस्तार से लिखें।

b)

Discuss usage of background Images, Colours and Properties in Cascading Style Sheets (CSS) documents.

कैस्केडिंग स्टाइल शीट्स (CSS) दस्तावेजों में पृष्ठभूमि छवियों, रंगों और गुणों के उपयोग पर चर्चा करें।

3. a)

Describe in detail about the Operators and Statements in JavaScript.

जावास्क्रिप्ट में ऑपरेटर्स और स्टेटमेंट्स के बारे में विस्तार से बताइए।

b)

Explain using loops in JavaScript with the help of an example.

एक उदाहरण की सहायता से जावास्क्रिप्ट में लूप के उपयोग को समझाइए।

4. a)

Discuss about the Features of XML documents. Define XML Objects.

XML दस्तावेजों की विशेषताओं के बारे में चर्चा करें। XML ऑब्जेक्ट को परिभाषित करें।

b)

Describe the concept of Transforming XML using XSL and XSLT using an example.

एक उदाहरण का उपयोग करके XSL और XSLT का उपयोग करके XML को बदलने की अवधारणा का वर्णन करें।

5. a)

Introduce AJAX. Describe AJAX Component with the help of an example.

AJAX का परिचय दें। एक उदाहरण की सहायता से AJAX घटक का वर्णन करें।

b)

Discuss about the need and usage of XMLHttpRequest Object using an example.

एक उदाहरण का उपयोग करके XMLHttpRequest ऑब्जेक्ट की आवश्यकता और उपयोग के बारे में चर्चा करें।

6. a)

Describe the concept of Manipulating Texts and Margins in HTML and Style Sheets.

HTML और स्टाइल शीट्स में टेक्स्ट और मार्जिन में हेरफेर की अवधारणा का वर्णन करें।

b)

Discuss the concept of Forms and Validations in JavaScript.

जावास्क्रिप्ट में फॉर्म और वैलिडेशन की अवधारणा पर चर्चा करें।

7. a)

Explain in detail about the XML Scripting using an example.

एक उदाहरण का उपयोग करके XML स्क्रिप्टिंग के बारे में विस्तार से बताइए।

b)

Write a detailed note on Web Services Vs Other technologies.

वेब सेवाओं बनाम अन्य तकनीकों पर एक विस्तृत नोट लिखें।

8. a)

Discuss in detail about the Web Services Benefits using suitable example.

उपयुक्त उदाहरण का उपयोग करके वेब सेवाओं के लाभों के बारे में विस्तार से चर्चा करें।

b)

Describe in detail about the Template Based Transformations in XML.

XML में टेम्पलेट आधारित परिवर्तनों के बारे में विस्तार से वर्णन करें।
