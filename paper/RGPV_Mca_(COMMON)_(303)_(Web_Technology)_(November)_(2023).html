Web Technology

Course: Mca Code: 303 Branch: COMMON Date: November 2023

Total No. of Questions : 8 Total No. of Printed Pages : 3
Roll No ........................

MCA-303 (2)

M.C.A. III Semester (Two Year Course)

Examination, November 2023

Web Technology

(Elective-I)

Time: Three Hours Maximum Marks: 70

Note: i) Attempt any five questions.

कोई पाँच प्रश्न हल करें।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जावेगा।

1. a)

Explain about the role of Internet in today's life. Explain Client/Server model.

आज के जीवन में इंटरनेट की भूमिका के बारे में बताइए। क्लाइन्ट / सर्वर मॉडल को समझाइये।

b)

With clear explanation describe the usage of WWW, IP and URL using an example.

स्पष्ट व्याख्या के साथ एक उदाहरण का उपयोग करके WWW, IP और URL के उपयोग का वर्णन करें।

2. a)

Write a detailed note on use of Hyperlinks and Tables in HTML.

HTML में हाइपरलिंक्स और टेबल्स के उपयोग पर एक विस्तृत नोट लिखें।

b)

Discuss XHTML, Meta tags in HTML using an example.

एक उदाहरण का उपयोग करके HTML में XHTML, मेटा टैग पर चर्चा करें।

3. a)

Describe in detail about the Client-side scripting with JavaScript using an example.

एक उदाहरण का उपयोग करके जावास्क्रिप्ट के साथ क्लाइंट-साइड स्क्रिप्टिंग के बारे में विस्तार से वर्णन करें।

b)

Explain Data Types and Variables in JavaScript with the help of an example.

जावास्क्रिप्ट में डाटा प्रकार और वैरिएबल्स को एक उदाहरण की सहायता से समझाइए।

4. a)

Discuss Features, Uses and Key Components of XML.

XML की विशेषताओं, उपयोगों और प्रमुख घटकों पर चर्चा करें।

b)

Describe markup elements and attributes in XML with the help of an example.

एक उदाहरण की सहायता से XML में मार्कअप तत्वों और विशेषताओं का वर्णन करें।

5. a)

Give a brief Introduction to AJAX. Describe AJAX Components.

AJAX का संक्षिप्त परिचय दें। AJAX घटकों का वर्णन करें।

b)

Explain using XSLT with AJAX with the help of an example.

एक उदाहरण की सहायता से AJAX के साथ XSLT का उपयोग समझाइए।

6. a)

Discuss in detail about the Web services and Web Service architecture in Web Technology.

वेब टेक्नोलॉजी में वेब सर्विसेज और वेब सर्विस आर्किटेक्चर के बारे में विस्तार से चर्चा करें।

b)

Describe in detail about the concept of effective navigation for your website with the help of an example.

एक उदाहरण की सहायता से अपनी वेबसाइट के लिए प्रभावी नेविगेशन की अवधारणा के बारे में विस्तार से वर्णन करें।

7. a)

Write a detailed note on Web Services Vs other technologies.

वेब सेवाओं बनाम अन्य तकनीकों पर एक विस्तृत नोट लिखें।

b)

Discuss about the need and usage of XMLHttpRequest Object using an example.

एक उदाहरण का उपयोग करके XMLHttpRequest ऑब्जेक्ट की आवश्यकता और उपयोग के बारे में चर्चा करें।

8. a)

Describe the concept of manipulating texts and margins in HTML and style sheets.

HTML और स्टाइल शीट्स में टेक्स्ट और मार्जिन में हेरफेर की अवधारणा का वर्णन करें।

b)

Discuss the concept of Forms and Validations in JavaScript.

जावास्क्रिप्ट में फॉर्म और वैलिडेशन की अवधारणा पर चर्चा करें।
