Object Oriented Programming With Java

Course: Mca Code: 205 Branch: MCA Date: June 2024

Save as PDF

Opens your browser print dialog — select "Save as PDF" to download.

Total No. of Questions : 8 Total No. of Printed Pages : 4
Roll No
MCA-205
M.C.A. II Semester (Two Year Course)
Examination, June 2024
Object Oriented Programming with JAVA
Time : Three Hours Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1. a)

Explain the concept of method Overriding and Overloading with an example. 7

विधि ओवरराइडिंग और ओवरलोडिंग की अवधारणा को उदाहरण सहित समझाइए। 7

b)

Describe in detail any three Object Oriented Programming principles. Illustrate with suitable examples. 7

ऑब्जेक्ट ओरिएंटेड प्रोग्रामिंग के किसी भी तीन सिद्धांतों का विस्तार से वर्णन करें। उपयुक्त उदाहरणों के साथ स्पष्ट करें। 7

2. a)

Why is Java considered to be platform independent? What is the role of Bytecode in making Java platform independent? 7

जावा को प्लेटफॉर्म स्वतंत्र क्यों माना जाता है? जावा को प्लेटफ़ॉर्म स्वतंत्र बनाने में बाइटकोड की क्या भूमिका है? 7

b)

Explain in detail about byte streams and character streams with suitable code samples. 7

उपयुक्त कोड नमूनों के साथ बाइट स्ट्रीम और कैरेक्टर स्ट्रीम के बारे में विस्तार से समझाइए। 7

3. a)

Describe in detail about exception handling, try block and catch clause with the help of a suitable Java program. 7

उपयुक्त जावा प्रोग्राम की सहायता से अपवाद हैंडलिंग, ट्राई ब्लॉक और कैच क्लॉज के बारे में विस्तार से वर्णन करें। 7

b)

Explain object streams in Java. Explain the role of Serializable interface with a suitable code sample. 7

जावा में ऑब्जेक्ट स्ट्रीम्स को समझाइए। उपयुक्त कोड नमूने के साथ Serializable इंटरफ़ेस की भूमिका को समझाइए। 7

4. a)

Explain in detail the Delegation Event model for event handling in Java. 7

जावा में इवेंट हैंडलिंग के लिए डेलीगेशन इवेंट मॉडल को विस्तार से समझाइए। 7

b)

Write a simple program by extending appropriate class to demonstrate the working of threads in Java. 7

जावा में थ्रेड्स की कार्यप्रणाली को प्रदर्शित करने के लिए उपयुक्त क्लास का विस्तार करके एक सरल प्रोग्राम लिखें। 7

5. a)

Describe in detail the creation of a thread using the Runnable interface and the Thread class with suitable examples. 7

उपयुक्त उदाहरणों के साथ Runnable इंटरफ़ेस और थ्रेड क्लास का उपयोग करके थ्रेड के निर्माण का विस्तार से वर्णन करें। 7

b)

Write a Java program to demonstrate the use of JLabel and JButton by adding them to JFrame. 7

JLabel और JButton को JFrame में जोड़कर उनके उपयोग को प्रदर्शित करने के लिए एक जावा प्रोग्राम लिखें। 7

6. a)

Write a client-server application using TCP Sockets where client can send message and server respond with the reverse of them. 7

TCP सॉकेट्स का उपयोग करते हुए एक क्लाइंट-सर्वर अनुप्रयोग लिखें, जहाँ क्लाइंट संदेश भेज सकता है और सर्वर उसके विपरीत प्रतिक्रिया दे सकता है। 7

b)

Write a Servlet that takes two numbers as an input and display all the prime numbers between them. Also write web.xml description for the servlet. 7

एक सर्वलेट लिखें जो दो संख्याओं को इनपुट के रूप में लेता है और उनके बीच की सभी अभाज्य संख्याएँ प्रदर्शित करता है। सर्वलेट के लिए web.xml विवरण भी लिखें। 7

7. a)

Explain JLabel component. With suitable examples explain any two of its constructors. 7

JLabel घटक को समझाइए। उपयुक्त उदाहरणों के साथ इसके किन्हीं दो कंस्ट्रक्टर्स को समझाइए। 7

b)

Explain step-by-step procedure of using Java Data Base Connectivity in Java programs. 7

जावा प्रोग्रामों में जावा डेटा बेस कनेक्टिविटी का उपयोग करने की चरण-दर-चरण प्रक्रिया समझाइए। 7

8. a)

Write short note on : 14

  • Struts Framework

    • अ) स्ट्रट्स फ्रेमवर्क

  • JSP

    • ब) JSP

  • GET and POST methods

    • स) GET और POST विधियाँ
