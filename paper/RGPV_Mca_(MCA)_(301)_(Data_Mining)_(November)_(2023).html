Data Mining

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.

a) Define Data mining. Explain different functionalities of data mining. (7)

डाटा माइनिंग को परिभाषित करें। डाटा माइनिंग की विभिन्न कार्यक्षमताओं की व्याख्या करें।

b) Explain different classifications of data mining systems. (7)

डाटा माइनिंग सिस्टम के विभिन्न वर्गीकरणों की व्याख्या करें।

2.

a) Discuss various OLAP operations with examples. (7)

उदाहरणों के साथ विभिन्न OLAP परिचालनों पर चर्चा करें।

b) Explain Multidimensional data model with suitable example. (7)

उपयुक्त उदाहरण के साथ बहुआयामी डाटा मॉडल को समझाए।

3.

a) What is Data cleaning? Describe the approaches to fill missing value. (7)

डाटा क्लीनिंग क्या है? तुम मान को भरने के तरीकों का वर्णन करें।

b) Explain different data mining primitives and languages. (7)

विभिन्न डाटा माइनिंग आदिम और भाषाओं की व्याख्या करें।

4.

a) Explain market basket analysis with suitable example. (7)

उपयुक्त उदाहरण के साथ मार्केट बास्केट विश्लेषण करें।

b) Explain constraint based association mining with example. (7)

बाधा-आधारित एसोसिएशन माइनिंग को उदाहरण सहित समझाए।

5.

Write Apriori algorithm and using the algorithm find all frequent item sets for the following databases. (14)

(Min_sup = 20%)

एप्रीओरी एल्गोरिथम लिखें और एल्गोरिथम का उपयोग करके निम्नलिखित डेटाबेस के लिए सभी लगातार आइटमसेट ढूंढें।

(न्यूनतम_सुप = 20%)

6.

a) Explain Navies Bayesian classification with example. (7)

नेवीज़ बायसियन वर्गीकरण को उदहारण सहित समझाए।

b) Give the different between classification and clustering methods. (7)

वर्गीकरण और क्लस्टरिंग विधियों के बीच अंतर बताएं।

7.

a) Explain different applications of Data mining. (7)

डाटा माइनिंग के विभिन्न अनुप्रयोगों की व्याख्या करें।

b) Explain decision tree induction with example. (7)

डिसिशन ट्री इंडक्शन को उदाहरण सहित समझाए।

8.

a) outlier analysis

b) prediction

c) clustering

अ) बाहरी विश्लेषण

ब) भविष्यवाणी

स) क्लस्टरिंग
