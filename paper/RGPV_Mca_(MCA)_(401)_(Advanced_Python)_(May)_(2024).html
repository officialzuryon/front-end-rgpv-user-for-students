MCA-401 (1)

M.C.A. IV Semester (Two Year Course)

Examination, May 2024

Advanced Python

(Elective-IV)

Time : Three Hours Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1. a)

Define Data Type. Explain various data types used to store data in Python.

डाटा टाइप परिभाषित करें। पायथन में डाटा को संग्रहित करने के लिए उपयोग किए जाने वाले विभिन्न डाटा टाइप की व्याख्या करें।

7
b)

Write a program to print characters from a string that are present at an even index number.

एक स्ट्रिंग से वर्णों को मुद्रित करने के लिए एक प्रोग्राम लिखें जो एक सम सूचकांक संख्या पर मौजूद हैं।

7
2. a)

Explain Logical and Bitwise operators in Python. Discuss short circuit policy for logical operators.

पायथन में लॉजिकल और बिटवाइज ऑपरेटर्स को समझाइए। लॉजिकल ऑपरेटर्स के लिए शॉर्ट सर्किट नीति पर चर्चा करें।

7
b)

Create a list by picking odd index items from the first list and even index items from the second.

पहली सूची से विषम अनुक्रमणिका आइटम और दूसरी से सम अनुक्रमणिका आइटम चुनकर एक सूची बनाए।

7
3. a)

Write a program code to create a vehicle class with max-speed mileage instance attributes and a method capacity as argument.

max-speed और माइलेज उदाहरण विशेषताओं और तर्क के रूप में क्षमता वाली एक विधि के साथ वाहन वर्ग बनाने के लिए एक प्रोग्राम कोड लिखें।

7
b)

Create a class 'Bus' that inherits from class 'vehicle' given in question 3(a). Give the capacity argument of Bus.seating-capacity a default value of 50.

एक क्लास 'बस' बनाएं जो प्रश्न 3(a) में दिये गये क्लास 'Vehicle' से विरासत में मिले। Bus.seating-capacity के क्षमता तर्क को 50 का डिफ़ॉल्ट मान दें।

7
4. a)

Write a python program to list only files in a directory.

किसी डायरेक्टरी में केवल फाइलों को सूचीबद्ध करने के लिए एक पायथन प्रोग्राम लिखें।

7
b)

Using question 4(a), handle the case where the directory does not exist or an error occur while accessing the directory.

प्रश्न 4(a) का उपयोग करते हुए, उस मामले को संभालें, जहाँ निर्देशिका मौजूद नहीं है या निर्देशिका तक पहुँचने के दौरान कोई त्रुटि उत्पन्न होती है।

7
5. a)

Explain working of SQLite module and its important methods to connect to SQLite database.

SQLite डाटाबेस से कनेक्ट करने के लिए इसके महत्वपूर्ण तरीकों के साथ SQLite मॉड्यूल की कार्यप्रणाली को समझाइए।

7
b)

Write the steps with a brief explanation to select rows from a SQLite table.

एक SQLite तालिका से पंक्तियों का चयन करने के लिए एक संक्षिप्त विवरण के साथ चरण लिखें।

7
6. a)

Give a brief comparison between class method, static method and instance method.

क्लास विधि, स्थिर विधि और इंस्टेंस विधि के बीच एक संक्षिप्त तुलना दें।

7
b)

Define constructor in Python. Also explain constructor overloading.

पायथन में कंस्ट्रक्टर को परिभाषित करें। कंस्ट्रक्टर ओव��लोडिंग भी समझाइए।

7
7. a)

Briefly explain the Exception model and the exception object in python.

पायथन में अपवाद मॉडल और अपवाद ऑब्जेक्ट को संक्षेप में समझाइए।

7
b)

Discuss how to handle multiple exceptions in Python.

चर्चा करें कि पायथन में एकाधिक अपवादों को कैसे प्रतिबंधित किया जाए।

7
8. a)

Explain the role of pandas in machine learning model building.

मशीन लर्निंग मॉडल निर्माण में पांडास की भूमिका की व्याख्या करें।

7
b)

Define Dynamic Visualization. Also write the steps to create a dynamic plot in Python.

गतिशील विजुअलाइजेशन को परिभाषित करें। पायथन में एक गतिशील प्लॉट बनाने के चरण भी लिखें।
