1. a)

Write a recursive algorithm for computing the nth Fibonacci number.

nth फिबोनांची संख्या की गणना के लिए एक पुनरावर्ती एल्गोरिथम लिखें।

2. a)

Generate recurrence relation from the recursive binary search algorithm.

पुनरावर्ती बाइनरी खोज एल्गोरिथम से पुनरावृति संबंध उत्पन्न करें।

b)

Consider the problem of finding the smallest and largest elements in an array n numbers using divide and Conquer method.

विभाजन और विजय विधि का उपयोग करके n संख्याओं की एक सरणी में सबसे छोटे और सबसे बड़े तत्वों को खोजने की समस्या पर विचार करें।

3. a)

Find the time complexity of the recurrence relation T(n) = n + T(n/10) + T(7n/10).

पुनरावृति संबंध T(n) = n + T(n/10) + T(7n/10) की समय जटिलता ज्ञात कीजिए।

b)

Analyze the time complexity of Strassen's algorithm compared to conventional matrix multiplication.

पारंपरिक मैट्रिक्स गुणन की तुलना में स्ट्रासेन के एल्गोरिथम की समय जटिलता का विश्लेषण करें।

b)

The recurrence T(n) = 7T (n/2) + n2 describe the running time of an algorithm A. A competing algorithm A has a running time of T (n) = aT (n/4) + n2. What is the largest integer value for a A' is asymptotically faster than A?

पुनरावृति T(n) = 7T (n/2) + n2 एक एल्गोरिथम A के चलने का समय वर्णित करता है। एक प्रतिस्पर्धी एल्गोरिथम A का चलने का समय T'(n) = aT (n/4) + n2 है। A' के लिए सबसे बड़ा पूर्णांक मान क्या है जो A से अनंतस्पर्शी रूप से तेज है?

द्विपद हीप के गुणों की व्याख्या करें। दो द्विपद हीप को एक करने के लिए एक एल्गोरिथम लिखें। और साथ ही, न्यूनतम कुंजी खोजने के लिए भी।

b)

Explain Backtracking. Let set S = {1, 3, 4, 5} and X= 8, we have to find subset sum problem using backtracking approach

बैकट्रैकिंग को समझाएं। मान लीजिए सेट S = {1, 3, 4, 5} और X= 8 है, हमें बैकट्रैकिंग दृष्टिकोण का उपयोग करके उपसमुच्चय योग समस्या ज्ञात करनी है।

b)
