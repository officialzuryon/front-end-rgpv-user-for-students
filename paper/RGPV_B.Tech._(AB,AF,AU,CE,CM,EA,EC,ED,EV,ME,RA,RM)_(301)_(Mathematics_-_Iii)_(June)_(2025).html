iii) In case of any doubt or dispute the English version question should be treated as final.

1.

a) If the equation f(x) is given as x³ - 2x - 5 = 0. Considering the initial approximation at x = 2 then find the value of next approximation correct upto 2 decimal places.

यदि समीकरण f(x) को x³ - 2x - 5 = 0 के रूप में दिया गया है। x = 2 पर प्रारंभिक सन्निकटन को ध्यान में रखते हुए, अगले सन्निकटन का मान 2 दशमलव स्थानों तक सही ज्ञात करें।

b) What is the rate of convergence of Newton Raphson method?

न्यूटन रेफ्सन विधि के अभिसरण की दर क्या है?

2.

a) Find

निम्नलिखित का मान ज्ञात करें।

i) A tan v

ii) A³e⁵

iii) A log x

b) Prove that f(4) = f(3) + ∆f(2) + ∆²f(1) + ∆³f(1) taking '1' as the interval of differencing.

सिद्ध कीजिए कि f(4) = f(3) + ∆f(2) + ∆²f(1) + ∆³f(1) अंतर के अंतराल के रूप में '1' लेते हुए।

3.

a) Using following table, by Lagrange's Method find/(x) as a poly nomial in x:

निम्नलिखित तालिका का उपयोग करते हुए, लेग्रेंज विधि द्वारा x में एक बहुपद के रूप में f(x) को खोजो।

b) Find f(3.5) from the following table using Newton's Forward interpolation formula.

न्यूटन के फॉरवर्ड इंटरपोलेशन सूत्र का उपयोग करके निम्न तालिका से f(3.5) खोजें।

a) Apply Gauss Elimination method to solve the following equations.

निम्नलिखित समीकरणों को हल करने के लिए गाउस उन्मूलन विधि लागू करें।

2x - y - 3z = 9

x - y - z = 6

x - y - z = 2

b) Find missing values in the following table

निम्नलिखित तालिका में लुप्त मान ज्ञात कीजिए।

5.

a) Find ∫₀¹ (1/(1+x)) dx approximately using Simpson's 3/8th rule on integration.

एकीकरण पर सिम्पसन के 3/8वें नियम का उपयोग करके ∫₀¹ (1/(1+x)) dx लगभग मूल्य ज्ञात कीजिए।

b) Solve the equation dy/dx = 1 - y with initial condition y(0) = 0 using Euler's modified method and tabulate the solution at x = 0.1, 0.2, 0.3.

यूलर संशोधित विधि का उपयोग करके प्रारंभिक स्थिति y(0) = 0 के साथ समीकरण dy/dx = 1 - y को हल करें और समाधान को x = 0.1, 0.2, 0.3 पर सारणीबद्ध करें।

a) Using Lagrange's interpolation formula to fit a polynomial to the data:

b) Using Regula-Falsi method, find the real root of xlog₁₀ x = 1.2, correct to four decimal places.

रेगुला-फाल्सी विधि का उपयोग करके, चार दशमलव स्थानों तक xlog₁₀ x = 1.2 का वास्तविक मूल ज्ञात करें।

7.

a) Find the inverse Laplace transform of (3s + 12) / (s² + 8s + 16)

का व्युत्क्रम लाप्लास रूपान्तरण ज्ञात कीजिए।

b) State convolution theorem and hence evaluate L⁻¹ [1 / (s(s² + a²))]

कनवोल्यूशन प्रमेय बताइए और इसलिए L⁻¹ [1 / (s(s² + a²))] का मूल्यांकन कीजिए।

8.

a) A binomial variable X Satisfies the relation 9P(X = 4) = P(X = 2) when n = 6. Find the value of the parameter p and P(X = 1).

एक द्विपद चर X सम्बन्ध 9P(X = 4) = P(X = 2) को सन्तुष्ट करता है जब n = 6 पैरामीटर p और P(X = 1) का मान ज्ञात करें।

b) A manufacturer knows from experience that the resistance of resistors he produces is with mean μ = 100Ω and s.d. σ = 2Ω. What percentage of resistors will have resistance between 98Ω and 102Ω?

एक निर्माता अनुभव से जानता है कि उसके द्वारा उत्पादित प्रतिरोधकों का प्रतिरोध औसत μ = 100Ω और s.d. σ = 2Ω है। कितने प्रतिशत प्रतिरोधकों का प्रतिरोध 98Ω और 102Ω के बीच होगा?
