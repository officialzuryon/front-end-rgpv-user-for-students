Internet And Web Technology

Course: B.Tech Code: 504 Branch: CS Date: December 2024

Total No. of Questions : 8 Total No. of Printed Pages : 3 [2]
Roll No.

CS-504 (A) (GS)

B.Tech., V Semester

Examination, December 2024

Grading System (GS)

Internet and Web Technology

Time : Three Hours Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जावेगा।

1.
a) Define Internet. What do you mean by Webpage? इंटरनेट को परिभाषित करें। वेबपेज से आप क्या समझते हैं?
b) What is a web server and a web browser? What is the interaction between a web browser and a web server? वेब सर्वर और वेब ब्राउजर क्या है? वेब ब्राउजर और वेब सर्वर के बीच क्या अंतरक्रिया होती है?
2.
a) What is DOM? Explain its usage with HTML with an example. डोम क्या है? HTML के साथ इसके उपयोग को एक उदाहरण सहित समझाएं।
b) Explain the various JavaScript objects and Host objects. विभिन्न जावा स्क्रिप्ट ऑब्जेक्ट और होस्ट ऑब्जेक्ट की व्याख्या करें।
3.
a) What is HTML? List the goals of SGML. Explain the various HTML tags to develop the web pages. HTML क्या है? SGML के लक्ष्यों की सूची बनाइए। वेब पेज विकसित करने के लिए विभिन्न HTML टैग की व्याख्या करें।
b) What is CSS? List out the various CSS properties. Explain the various concepts of CSS properties with neat example. CSS क्या है? विभिन्न CSS गुणों की सूची बनाइए। CSS गुणों की विभिन्न अवधारणाओं को स्पष्ट उदाहरण के साथ समझाइए।
4.
a) What is a DTD (Document Type Definition). What are its applications? DTD (दस्तावेज़ प्रकार परिभाषा) क्या है? इसके अनुप्रयोग क्या हैं?
b) Discuss Web design issues in details. वेब डिज़ाइन के मुद्दों पर विस्तार से चर्चा करें।
5.
a) Define XHTML. What are the differences between XHTML and HTML with an example. XHTML को परिभाषित करें। उदाहरण के साथ XHTML और HTML के बीच क्या अंतर है।
b) How can both Internal and External DTDs be used in an XML File? Show with an example. XML फ़ाइल में आंतरिक और बाहरी दोनों DTD का उपयोग कैसे किया जा सकता है? एक उदाहरण सहित दिखाएं।
6.
Explain the following terms: निम्नलिखित शब्दों को स्पष्ट कीजिए
  1. Cookies and sessions
  2. Creating Database
  3. Planning and Publishing Websites
7.
a) List the statements that are used to connect PHP with MySQL with an example. PHP को MySQL से जोड़ने के लिए उपयोग किए जाने वाले कथनों को एक उदाहरण सहित सूचीबद्ध करें।
b) Explain the various events handlers in java script. Give an example of each. जावा स्क्रिप्ट में विभिन्न इवेंट हैंडेलर्स को समझाइए। प्रत्येक का एक उदाहरण दें।
8.
Write a short notes: संक्षिप्त नोट्स लिखें :
  1. XML Key Components
  2. Object oriented programming in PHP
  3. MySQL
  4. Need for CSS
