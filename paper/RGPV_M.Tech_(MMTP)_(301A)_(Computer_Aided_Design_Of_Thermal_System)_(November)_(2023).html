MMTP-301(A)
M.E./M.Tech., III Semester
Examination, November 2023
Computer Aided Design of Thermal System
(Elective-I)
Time: Three Hours Maximum Marks: 70

Note:

  1. Attempt any five questions.
  2. All questions carry equal marks.
  1. 1. a)
    Define Thermal systems and how it is significant and write merits and demerits?
    b)
    Write in detail about design considerations and explain with involved parameters.
  2. 2. a)
    Explain various types of design problems with all possible figures.
    b)
    Discuss processes involved in conceptual designs.
  3. 3. a)
    Write about the modeling and explain it in detail.
    b)
    Give in detail about the features of thermal designs.
  4. 4. a)
    What are thermal processes and explain why and how it is advantageous?
    b)
    Describe in detail about dimensional analysis and how it affects the system?
  5. 5. a)
    Define a concept simulation and explain how it is significant during simulation?
    b)
    Classify modeling techniques and explain that in full length.
  6. 6. a)
    Discuss numerical simulation and modeling explain them in detail.
    b)
    Write about the system simulation with suitable sketches.
  7. 7. a)
    State optimization techniques and explain their features.
    b)
    What are the constraints and discuss them in detail?
  8. 8.
    Write a short note on any two:
    1. Linear Programming
    2. Dynamic programming
    3. Calculus Method
