SM/2025/8106

Second Year Pharmacy

PHARMACOLOGY

(ER20-21T)

Time : Three Hours

Maximum Marks : 80

Note :

  1. This Paper is having three sections
  2. All sections are compulsory
  3. Attempt questions as per instructions given in each section

Section-A

(Objective Type Questions)

20 × 1 = 20

Note : Answer all 20 questions (Multiple Choice Questions / Fill in the Blanks / One Word / One sentence (Questions) Each question carries one mark.

1.
An antihypertensive agents which is safe in pregnancy is _________
2.
_________ is an example of Mucolytic agent
3.
The local anaesthetic which is widely used in Dental surgery is _________
4.
DOTS stands for _________
5.
Ambroxol is primarily given for _________
6.
PPI stands for _________
7.
_________ is an example of Loop diuretics
8.
ACE stands for _________
9.
Name the branch of Pharmacology dealing with the poisonous effects of drugs
10.
What is Bioavailability?
11.
Name the test employed for the detection of Typhoid infection.
12.
What is Juvenile Diabetes?
13.
Give an example for tricyclic antidepressant
14.
Statins are antagonist of which enzyme during elevated blood cholesterol level.
15.
Name a non-benzodiazepine hypnotic.
16.
Mention an example for anti-thyroxine drug.
17.
Name 2 example of calcium channel blocker used in hypertension
18.
Give an example for an intermediate acting non-depolarizing neuromuscular blocking agent
19.
Give 2 example of Thrombolytic agent.
20.
What is the antidote for Atropine poisoning?

Section-B

(Short Answer Type Questions)

10 × 3 = 30

Note : Answer any ten questions out of eleven. Each question carries three marks.

1.
Explain the symptoms and management of COPD.
2.
Classify Anti-diarrheal drugs with examples
3.
Discuss the mechanism of action and therapeutic uses of Penicillin
4.

11.

(Long Answer Type Questions)

6 × 5 = 30

Note : Answer any six questions out of seven. Each question carries five marks.
