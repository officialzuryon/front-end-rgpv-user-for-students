PHARMACOGNOSY

(ER20-13T)

Time : Three Hours Maximum Marks : 80
Note :
i) This question paper contain three parts.
ii) Part-A : Solve all twenty question. Each question carry one marks.
iii) Part-B : Short answer type questions contain eleven questions, attempt any ten question. Each question carry three marks.
iv) Part-C : Long answer type questions contain seven questions, attempt any six questions. Each question carry five marks.
