1.
a) Define Embedded system. Explain application areas of embedded system

b) Give a brief classification of embedded system Discuss its characteristics and quality attributes.
एम्बेडेड सिस्टम का संक्षिप्त वर्गीकरण दें। इसकी विशेषताओं और गुणवत्ता गुणों पर चर्चा करें।
2.
a) Give a brief introduction on Von Neumann V/s Harvard Architecture
वॉन न्यूमैन बनाम हार्वर्ड आर्किटेक्चर का संक्षिप्त परिचय दें।
b) Explain the following with respect to embedded system
i) ARM
ii) DSP Processor
एम्बेडेड सिस्टम के संबंध में निम्नलिखित की व्याख्या करें।
i) ARM
ii) DSP प्रोसेसर

b) Explain about the basic instruction set architecture in embedded system.
एम्बेडेड सिस्टम में बेसिक इंस्ट्रक्शन सेट आर्किटेक्चर के बारे में बताएं।
