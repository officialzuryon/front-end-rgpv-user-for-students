सही उत्तर का चयन कीजिए।

i) What is the difference between a disaster and a hazard?
  1. A disaster is natural, while a hazard is man-made.
  2. A hazard indicates potential damage, while a disaster is an actual event of damage.
  3. Both are the same
  4. None

आपदा और खतरे (Hazard) में क्या अंतर है?

  1. आपदा प्राकृतिक होती है, जबकि खतरा मानवजनित होता है
  2. खतरा संभावित नुकसान का संकेत देता है, जबकि आपदा नुकसान की वास्तविक घटना होती है
  3. दोनों समान हैं
  4. कोई नहीं
ii) Tsunami is an example of which type of disaster?
  1. Geological
  2. Biological
  3. Man-made
  4. Technological

त्सुनामी किस प्रकार की आपदा है?

  1. भूवैज्ञानिक
  2. जैविक
  3. मानवजनित
  4. तकनीकी
iii) What is the primary activity during a disaster?
  1. Risk mapping
  2. Rehabilitation
  3. Relief and rescue operations
  4. Mitigation strategies

आपदा के दौरान कौन-सी प्राथमिक गतिविधि होती है?

  1. जोखिम मानचित्रण
  2. पुनर्वास
  3. राहत और बचाव कार्य
  4. शमन रणनीति
iv) Which of the following government institutions is responsible for disaster management in India?
  1. NDRF
  2. ISRO
  3. RBI
  4. SEBI

निम्नलिखित में से कौन-सा सरकारी संस्थान आपदा प्रबंधन में कार्य करता है?

  1. एन.डी.आर.एफ.
  2. इसरो
  3. आर.बी.आई.
  4. सेबी
v) What is the primary purpose of remote sensing?
  1. Predict earthquakes
  2. Monitor and analyze disasters
  3. Create government policies
  4. Store financial data

रिमोट सेंसिंग का मुख्य उद्देश्य क्या है?

  1. भूकंप की भविष्यवाणी करना
  2. आपदाओं की निगरानी और विश्लेषण करना
  3. सरकारी योजनाएँ बनाना
  4. वित्तीय डेटा संग्रहित करना
