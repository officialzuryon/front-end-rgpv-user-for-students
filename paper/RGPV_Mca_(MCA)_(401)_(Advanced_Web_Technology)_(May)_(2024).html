Advanced Web Technology

Course: Mca Code: 401 Branch: MCA Date: May 2024

Save as PDF

Total No. of Questions : 8 Total No. of Printed Pages : 3
Roll No.

MCA-401 (2)

M.C.A. IV Semester (Two Year Course)

Examination, May 2024

Advanced Web Technology

(Elective-IV)

Time : Three Hours Maximum Marks : 70

Note: i) Attempt any five questions.

किसी पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

  1. 1. a)

    Explain the elements required for responsive web design.

    रिस्पॉन्सिव वेब डिजाइन के लिए आवश्यक तत्वों की व्याख्या करें।

    7
    b)

    Give the basic structure of a Bootstrap Grid and briefly explain Grid classes.

    बूटस्ट्रैप ग्रिड की मूल संरचना दें और ग्रिड कक्षाओं की संक्षेप में व्याख्या करें।

    7
  2. 2. a)

    Compare and contrast between XML and HTML.

    XML और HTML के बीच तुलना करें और अंतर बताइए।

    7
    b)

    Explain well formed XML document rules with a suitable XML code.

    एक उपयुक्त XML code के साथ सुपरव्यवस्थित XML डॉक्यूमेंट नियमों की व्याख्या करें।

    7
  3. 3. a)

    Explain AJAX web application model and show the working of AJAX.

    AJAX वेब एप्लीकेशन मॉडल की व्याख्या करें और AJAX की कार्यप्रणाली दिखाएँ।

    7
    b)

    Explain the following XSLT functions with syntax

    सिंटेक्स के साथ निम्नलिखित XSLT फंक्शंस को समझाइए।

    1. current()
    2. key()
    3. function-available()
    7
  4. 4. a)

    Create a Javascript code to send data (student_name) to the server using AJAX.

    AJAX का उपयोग करके सर्वर पर डाटा (student_name) भेजने के लिए एक Javascript कोड बनाइए।

    7
    b)

    Create a PHP code that processes the data received on server side in question 4(a).

    PHP कोड बनाएँ जो प्रश्न 4(a) में सर्वर साइड पर डाटा को संसाधित करता है।

    7
  5. 5. a)

    Suppose you need to send user_id, password, name, email to the server. So, create a Javascript function to send this data in XML format to the server.

    मान लीजिए आपको सर्वर पर user_id, पासवर्ड, नाम, ईमेल भेजने की आवश्यकता है। इसलिए इस डाटा को XML प्रारूप में सर्वर पर भेजने के लिए एक Javascript फ़ंक्शन बनाइए।

    7
    b)

    Write a code in PHP to collect the data sent through AJAX request in question 5(a) and save the data in an XML file.

    प्रश्न 5(a) में AJAX अनुरोध के माध्यम से भेजे गए डाटा को एकत्र करने के लिए PHP में एक कोड लिखें और डाटा को XML फाइल में सेव करें।

    7
  6. 6. a)

    Explain the role of UDDI in web services.

    वेब सेवाओं में UDDI की भूमिका स्पष्ट करें।

    7
    b)

    Discuss the SOAP web services with its advantages and disadvantages.

    SOAP वेब सेवाओं पर इसके फायदे और नुकसान के साथ चर्चा करें।

    7
  7. 7. a)

    Compare and contrast SOAP and REST protocols.

    SOAP और REST प्रोटोकॉल की तुलना करें और अंतर समझाइए।

    7
    b)

    Explain Javascript DOM objects with their methods to access the HTML elements.

    HTML तत्वों तक पहुँचने के तरीकों के साथ Javascript DOM objects की व्याख्या करें।

    7
  8. 8. a)

    Write a Javascript code to test if jQuery is loaded.

    यह जाँचने के लिए कि jQuery लोड है या नहीं, एक Javascript कोड लिखें।

    7
    b)

    Explain jQuery methods to access HTML attributes.

    HTML विशेषताओं तक पहुँचने के लिए jQuery तरीकों की व्याख्या करें।

    7
