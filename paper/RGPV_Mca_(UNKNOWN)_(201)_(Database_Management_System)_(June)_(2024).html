MCA-201

M.C.A. II Semester (Two Year Course)

Examination, June 2024

Database Management System

Time : Three Hours

Maximum Marks : 70

Note:

  1. Attempt any five questions.
  2. किन्हीं पाँच प्रश्नों को हल कीजिए।
  3. All questions carry equal marks.
  4. सभी प्रश्नों के समान अंक हैं।
  5. In case of any doubt or dispute the English version question should be treated as final.
  6. किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।
1.

a) State four advantages of DBMS over file processing system.

फाइल प्रसंस्करण प्रणाली की तुलना में DBMS के चार लाभ बताइये।

b) Draw ER diagram for Library Management System with at least 4 entities and one weak entity.

कम से कम 4 इकाईयों और एक कमजोर इकाई के साथ पुस्तकालय प्रबंधन प्रणाली के लिए ER आरेख बनाइए।

2.

a) Consider a schema R (A, B, C, D) and functional dependencies A → B and C → D. Solve and find whether the decomposition of R into R1 (A, B) and R2 (C, D) belongs to which one or both (dependency preserving and loss less join)?

स्कीमा R (A, B, C, D) और कार्यात्मक निर्भरता A → B और C → D पर विचार करें। हल करें और पता लगाएँ कि R का R1 (A, B) और R2 (C, D) में विघटन किससे संबंधित है या दोनों से (निर्भरता संरक्षण और हानि रहित संयोजन)?

b) Define Join operation. Explain different types of joins?

जोइन ऑपरेशन को परिभाषित करें। जोइन के विभिन्न प्रकारों को समझाइए।

3.

a) Let EMP (empno, deptno, ename, salary, Designation, DOB, city) be a table. Write the SQL query to perform the following.

  1. Insert one row into the table
  2. Delete the employee whose designation is manager and who lives at Delhi.
  3. Display the list of the employees who earn the maximum salary.
  4. Add one attribute date_of_joining in the table EMP.
  5. Display employee name staring with "S" and working in deptno 201.

मान लें कि EMP (empno, deptno, ename, salary, Designation, DOB, city) एक टेबल है। निम्नलिखित कार्य करने के लिए SQL क्वेरी लिखें।

  1. टेबल में एक पंक्ति डालें।
  2. उस कर्मचारी को हटाएँ जिसका पदनाम प्रबंधक है और जो दिल्ली में रहता है।
  3. अधिकतम वेतन पाने वाले कर्मचारियों की सूची प्रदर्शित करें।
  4. टेबल EMP में एक विशेषता date_of_joining जोड़ें।
  5. कर्मचारी का नाम "S" से शुरू करके और deptno 201 में काम करते हुए प्रदर्शित करें।
[3] [4]
4.

a) Write a PL/SQL program to check whether specified employee is present in EMP table or not. Accept empno from user. If employee does not exist display message using exception handling.

यह जाँचने के लिए PL/SQL प्रोग्राम लिखें कि निर्दिष्ट कर्मचारी EMP तालिका में मौजूद है या नहीं। उपयोगकर्ता से empno स्वीकार करें। यदि कर्मचारी मौजूद नहीं है तो अपवाद हैंडलिंग का उपयोग करके संदेश प्रदर्शित करें।

b) Define Normalization and state three advantages of Normalization.

सामान्यीकरण को परिभाषित करें तथा सामान्यीकरण के तीन लाभ बताइए।

b) Differentiate between :

  1. 3 NF and BCNF
  2. Trivial and Non-trivial Multivalued Dependency

अंतर करें :

  1. 3 NF और BCNF
  2. तुच्छ और गैर-तुच्छ बहुमूल्य निर्भरता
5.

a) Explain steps of cursor implementation with syntax and example.

सिंटैक्स और उदाहरण के साथ कर्सर कार्यान्वयन के चरणों को समझाइए।

b) Explain the concept of data replication and data fragmentation.

डाटा प्रतिकृति और डाटा विखंडन की अवधारणा को समझाइए।

6.

a) Explain different transaction states with a neat diagram.

अलग-अलग लेन-देन स्थितियों को एक साफ चित्र के साथ समझाइए।

b) Explain the ACID properties of a transaction.

किसी लेनदेन के ACID गुणों की व्याख्या करें।

7.

a) Discuss the different types of indexes.

विभिन्न प्रकार के सूचकांकों पर चर्चा कीजिए।

b) Write short notes on database backup and database recovery.

डाटाबेस बैकअप और डाटाबेस रिकवरी पर संक्षिप्त नोट्स लिखें।

8.

Write short notes on any two :

  1. Data Warehousing
  2. Network and Hierarchical model
  3. RAID

किन्हीं दो पर संक्षिप्त टिप्पणी लिखें।

  1. डाटा वेयरहाउसिंग
  2. नेटवर्क और पदानुक्रमित मॉडल
  3. RAID
