Operating System

Note : (i) Attempt any five questions.

किसी पाँच प्रश्नों को हल कीजिए।

सभी प्रश्नों के समान अंक हैं।

1. (a)

Define an Operating System. Explain its objectives and major services provided by an OS.

ऑपरेटिंग सिस्टम को परिभाषित करें। इसके उद्देश्यों एवं OS द्वारा प्रदान की जाने वाली प्रमुख सेवाओं की व्याख्या करें।

(b)

Describe different types of Operating Systems with suitable examples.

उपयुक्त उदाहरणों सहित विभिन्न प्रकार के ऑपरेटिंग सिस्टम का वर्णन करें।

2. (a)

Explain the structure of an Operating System. Discuss system calls and their importance.

ऑपरेटिंग सिस्टम की संरचना की व्याख्या करें। सिस्टम कॉल्स एवं उनके महत्व पर चर्चा करें।

(b)

Explain OS design and implementation issues. What is spooling and buffering?

3. (a)

Explain CPU scheduling criteria and different CPU scheduling algorithms.

CPU शेड्यूलिंग मापदंडों एवं विभिन्न CPU शेड्यूलिंग एल्गोरिथम की व्याख्या करें।

(b)

Calculate the average waiting time using FCFS scheduling

दिए गए प्रोसेस के लिए FCFS शेड्यूलिंग का उपयोग करते हुए औसत प्रतीक्षा समय ज्ञात करें।

4. (a)

Explain process concept and operations on processes.

प्रोसेस की अवधारणा एवं प्रोसेस पर की जाने वाली क्रियाओं की व्याख्या करें।

(b)

Explain critical section problem and semaphores.

क्रिटिकल सेक्शन समस्या एवं सेमाफोर की व्याख्या करें।

5. (a)

Explain the deadlock and its necessary conditions.

डेडलाक की व्याख्या करें तथा इसकी आवश्यक शर्तें बताइए।

(b)

Describe deadlock prevention, avoidance, and detection techniques.

डेडलाक प्रिवेंशन, अवॉइडेन्स एवं डिटेक्शन तकनीकों का वर्णन करें।

6. (a)

Explain memory management techniques including fixed and dynamic partitioning.

फिक्स्ड एवं डायनामिक पार्टिशनिंग सहित मेमोरी मैनेजमेंट तकनीकों की व्याख्या करें।

(b)

Given memory blocks of sizes 100 KB, 500 KB, 200 KB, 300 KB, and allocate a process of size 212 KB using First-Fit and Best-Fit algorithms.

दिए गए मेमोरी ब्लॉक्स के लिए 212 KB प्रोसेस को First-Fit एवं Best-Fit एल्गोरिथम द्वारा अलॉट करें।

7. (a)

पेज रिप्लेसमेंट एल्गोरिथम एवं थ्रेशिंग की अवधारणा समझाइए।

8. (a)

Explain disk scheduling and different disk scheduling algorithms.

डिस्क शेड्यूलिंग एवं विभिन्न डिस्क शेड्यूलिंग एल्गोरिथम की व्याख्या करें।

(b)

Explain file management in Operating System. Discuss file organization and access methods.

ऑपरेटिंग सिस्टम में फाइल मैनेजमेंट की व्याख्या करें। फाइल ऑर्गनाइजेशन एवं एक्सेस मेथड्स पर चर्चा करें।
