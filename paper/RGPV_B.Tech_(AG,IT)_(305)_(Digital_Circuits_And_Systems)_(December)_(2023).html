Design a 4-bit BCD to Excess-3 code convertor and realize it using 2-level gate implementation.
एक 4-बिट BCD से एक्सेस-3 कोड कन्वर्टर डिज़ाइन करें और 2-स्तरीय गेट कार्यान्वयन का उपयोग करके इसे साकार करें।
2.
Design a digital circuit which converts a 3-bit binary number into its equivalent 1's compliment form.
एक डिजिटल सर्किट डिजाइन करें जो 3-बिट बाइनरी नंबर को उसके समतुल्य 1's के कॉम्प्लिमेंट फॉर्म में परिवर्तित करता है।
Draw the logic diagram of an 8-to-3 line encoder. Explain its operation using truth table.
8-टू-3 लाइन एन्कोडर का लॉजिक आरेख बनाइए। सत्य सारणी का उपयोग करके इसकी कार्यप्रणाली समझाइए।
3.
Explain the operation of a priority encoder. Design and draw the logic diagram of the same.
प्रायोरिटी एन्कोडर के संचालन की व्याख्या करें। उसका लॉजिक डायग्राम डिजाइन करें और बनाइए।
Implement the following Boolean functions using 4 × 1 multiplexer and external logic gates. Connect inputs 'A' and 'C' to select lines.
F(A, B, C, D) = Σ(0, 3, 5, 6, 8, 9, 11, 13, 15)
निम्नलिखित बूलियन फंक्शन्स को कार्यान्वित करें 4 × 1 मल्टीप्लेक्सर और बाहरी लॉजि�� गेट्स का उपयोग करके। इनपुट 'A' और 'C' को चयनित लाइनों से कनेक्ट करें।
F(A, B, C, D) = Σ(0, 3, 5, 6, 8, 9, 11, 13)
4.
Implement a full adder using a decoder and NAND gates only. The inputs to the adder are 'A', 'B' and 'C' and the outputs are 'S' and 'Cout'.
केवल डिकोडर और NAND गेट्स का उपयोग करके पूर्ण योजक को कार्यान्वित करें। एडर में इनपुट 'A', 'B' और 'C' हैं और आउटपुट 'S' और 'Cout' हैं।
Design a full subtractor using only required number of half subtractors.
केवल आधे सबट्रैक्टर की आवश्यक संख्या का उपयोग करके एक पूर्ण सबट्रैक्टर डिज़ाइन करें।
5.
Convert a D flip-flop into J-K flip-flop. Show all the necessary calculations with final logic diagram. Use only 2 input NAND gates to realise any Boolean function required for this conversion.
D फ्ल��प-फ्लॉप को J-K फ्लिप-फ्लॉप में परिवर्तित करें। अंतिम तर्क आरेख के साथ सभी आवश्यक गणना दिखाएं। इस रूपांतरण के लिए आवश्यक किसी भी बूलियन फंक्शन को प्राप्त करने के लिए केवल 2-इनपुट NAND गेट का उपयोग करें।
Design an asynchronous MOD-4 DOWN counter using T flip-flop.
T फ्लिप-फ्लॉप का उपयोग करके एक असिंक्रोनस MOD-4 DOWN काउंटर डिजाइन करें।

यूनिवर्सल शिफ्ट रजिस्टर का लॉजिक आरेख बनाइए। मान लें (S1, S0) मोड नियंत्रण पिन हैं, जैसे कि रजिस्टर ऑपरेशन के निम्नलिखित मोड का निरीक्षण करता है।

7.
Design a 2-input NOR gate using PMOS logic and explain its operation.
PMOS लॉजिक का उपयोग करके 2-इनपुट NOR गेट डिजाइन करें और इसके संचालन की व्याख्या करें।

JK फ्लिप-फ्लॉप का उपयोग करके निम्नलिखित अनुक्रम की गणना करने के लिए एक सिंक्रोनस काउंटर डिजाइन करें।
