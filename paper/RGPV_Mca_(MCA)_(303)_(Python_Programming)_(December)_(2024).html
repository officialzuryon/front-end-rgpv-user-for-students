MCA-303 (1)

M.C.A. III Semester (Two Year Course)

Examination, December 2024

Python Programming

(Elective-I)

Time : Three Hours Maximum Marks : 70

Note:

  • i) Attempt any five questions.

    किन्हीं पाँच प्रश्नों को हल कीजिए।

  • ii) All questions carry equal marks.

    सभी प्रश्नों के समान अंक हैं।

  • iii) In case of any doubt or dispute the English version question should be treated as final.

    किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.

a) Explain different data types supported by Python.

पायथन द्वारा समर्थित विभिन्न डाटा प्रकारों को समझाइये।

b) Explain any six arithmetic operators of Python with suitable example of each.

पायथन के किन्हीं छह अंकगणितीय ऑपरेटरों को प्रत्येक के उपयुक्त उदाहरण के साथ समझाइए।

2.

a) Write a Python program to calculate distance between two points.

दो बिंदुओं के बीच की दूरी की गणना करने के लिए पायथन प्रोग्राम लिखें।

3.

a) Write a Python program to swap the given two numbers.

दी गई दो संख्याओं को बदलने के लिए पायथन प्रोग्राम लिखें।

b) Explain the purpose of loop statement in a programming language. Describe the syntax and semantics of loop statements provided by Python with the help of suitable example.

प्रोग्रामिंग भाषा में लूप स्टेटमेंट के उद्देश्य को समझाइए। उपयुक्त उदाहरण की सहायता से पायथन द्वारा प्रदान किए गए लूप स्टेटमेंट के सिंटैक्स और शब्दार्थ का वर्णन कीजिए।

b) Compare and contrast the difference between List and Arrays and also explain array module with example programs.

सूची और सारणी के बीच अंतर की तुलना करें और उदाहरण कार्यक्रमों के साथ सारणी मॉड्यूल के बारे में भी समझाइए।

4.

a) Write a Python program to accept three sides of triangle and check whether the triangle is possible or not.

त्रिभुज की तीन भुजाओं को स्वीकार करने और यह जाँचने के लिए कि त्रिभुज संभव है या नहीं, पायथन प्रोग्राम लिखें।

b) Write a Python program to find largest of three numbers using decision statement.

निर्णय कथन का उपयोग करके तीन संख्याओं में से सबसे बड़ी संख्या ज्ञात करने के लिए पायथन प्रोग्राम लिखें।

5.

a) Write a Python program to store 'n' numbers in a list and sort the list using insertion sort.

एक सूची में 'n' संख्याओं को संग्रहीत करने और प्रविष्टि सॉर्ट का उपयोग करके सूची को सॉर्ट करने के लिए एक पायथन प्रोग्राम लिखें।

b) Compare and contrast tuples and list in Python.

पायथन में ट्यूपल और सूची की तुलना करें और अंतर करें।

6.

a) How to copy one dictionary to another? Explain different ways to copy dictionary with example.

एक शब्दकोश को दूसरे में कैसे कॉपी करें? शब्दकोश की प्रतिलिपि बनाने के विभिन्न तरीकों को उदाहरण सहित समझाइए।

b) Write a Python program to read the list of numbers. Also display 1+1 sum and average.

संख्याओं की सूची पढ़ने के लिए पायथन प्रोग्राम लिखें। 1+1 योग और औसत भी प्रदर्शित करें।

7.

a) Explain classes and objects in Python with suitable program.

उपयुक्त प्रोग्राम के साथ पायथन में क्लासेस और ऑब्जेक्ट्स को समझाइए।

b) Discuss constructor and destructor in Python with suitable program.

उपयुक्त प्रोग्राम के साथ पायथन में कंस्ट्रक्टर और डिस्ट्रक्टर पर चर्चा करें।

8.

a) What is Exception Handling? Discuss different types of exception in Python.

अपवाद प्रबंधन क्या है? पायथन में अपवाद के विभिन्न प्रकारों पर चर्चा करें।

b) Write a function in Python to read the content from a text file "rgvp.txt" line by line and display the same on screen.

टेक्स्ट फ़ाइल "rgvp.txt" की सामग्री को लाइन दर लाइन पढ़ने और स्क्रीन पर प्रदर्शित करने के लिए पायथन में एक फ़ंक्शन लिखें।
