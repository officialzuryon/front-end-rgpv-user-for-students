Final Year Pharmacy

PHARMACEUTICAL CHEMISTRY-II

(202)

Time : Three Hours Maximum Marks : 80

Note : Attempt any six questions. Question No. 1 is compulsory.

1.
Attempt any five questions from (a) and (b). 5×2=10
a)
Write the chemical structure of following.
  1. Butoxy
  2. Vinyl
  3. Thiophene
  4. Tropane
  5. Barbituric Acid
  6. Phenoxy
b)
Write the chemical name of followings.
  1. CH3CONH-
  2. H2NSO2
  3. H2NCH2CO-
2.
Write down the chemical structure, action and uses with pharmaceutical preparation, Brand name of followings. (any two) 2×7=14
  1. Non-adrenaline
  2. Halothane
  3. Tolbutamide
3.
Define hypnotics and sedatives. Discuss any one Benzodiazepine derivative of this class. 14
4.
Write down the note on (any two) 2×7=14
  1. Cardiovascular agents
  2. Analeptics
  3. Diagnostic agents
5.
Define and classify Antiseptic and Disinfectant Agents, discuss properties, chemical structure, brand name and uses of any two drugs. 14
6.
Give detail about one drug of each category (any two) 2×7=14
  1. Quinoline derivative as Antimalarial
  2. Guanidine derivative as oral Hypoglycemic agents
  3. Thiouracil derivative as antithyroid
7.
Write brief notes on any two. 2×7=14
  1. Antileprotic agents
  2. NSAIDs
  3. Local Anaesthetics
8.
Define Parasympathetic agents. Write down structure, properties, uses and official compounds of any two drugs. 14
