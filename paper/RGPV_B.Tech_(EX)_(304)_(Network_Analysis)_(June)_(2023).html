EX-304 (GS)

B.Tech., III Semester

Examination, June 2023

Grading System (GS)

Network Analysis

Time : Three Hours

Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जावेगा।

1.

a) State and explain Kirchhoff's voltage and current law with an example.

उदाहरण के साथ किरचॉफ के वोल्टेज और करंट कानून को बताएँ और समझाएँ।

b) Define Graph, Tree, Basic tie set matrix and cut set matrix for a planar network with an example.

उदाहरण के साथ प्लानर नेटवर्क के लिए ग्राफ, ट्री, बेसिक टाई सेट मैट्रिक्स और कट सेट मैट्रिक्स को परिभाषित करें।

2.

a) For a series resonant circuit with constant voltage and variable frequency, obtain the frequency at which voltage across the inductor is maximum. Calculate this maximum voltage when R=50 ohms, L=0.05H, C=20µF and V=100 volts.

निसेलर वोल्टेज और पर आवृत्ति के साथ एक श्रृंखला अनुनाद सर्किट के लिए, आवृत्ति प्राप्त करें जिस पर प्रारंभ करने वाला में वोल्टेज अधिकतम है। अधिकतम वोल्टेज की गणना करें जब R=50 ohms, L=0.05H, C=20 माइक्रो फैरड और V=100 वोल्ट।

b) Find the current 'I' by using Norton's Theorem.

नॉर्टन के प्रमेय का उपयोग करके वर्तमान 'I' का पता लगाएँ।

नीचे दिए गए चित्र में दिखाए गए सर्किट के लिए, टर्मिनल AB पर थेवेनिन समतुल्य सर्किट बनाएँ और फिर प्रतिरोचक भार का मान ज्ञात करें, ताकि इसे अधिकतम शक्ति प्राप्त हो। अधिकतम शक्ति का मान भी पता लगाएँ।

b) For the waveform shown in figure below. Write the expression and find the Laplace transform.

नीचे दिए गए चित्र में दिखाए गए तरंग के लिए व्यंजक लिखिए और लाप्लास रूपांतरण ज्ञात कीजिए।

4.

a) In the circuit given below the switch is closed at t=0. By using Laplace transform find the current i(t).

नीचे दिए गए परिपथ में स्विच t=0 पर बंद है। लैपलेस ट्रांसफॉर्म का उपयोग करके करंट i(t) ज्ञात कीजिए।

b) State and explain the Tellegen's theorem with suitable example.

उपयुक्त उदाहरण के साथ टेलेजेन के प्रमेय को लिखिए और समझाएँ।

5.

a) State and explain Initial Value and Final Value theorem.

प्रारंभिक मूल्य और अंतिम मूल्य प्रमेय को बताएँ और समझाएँ।

b) Draw the pole Zero diagram for the given network function I(s) and hence obtain i(t)

दिए गए नेटवर्क फंक्शन I(s) के लिए ध्रुव-शून्य आरेख बनाएँ और i(t) ज्ञात कीजिए।

I(s) = 20s/[(s+5)(s+2)]
