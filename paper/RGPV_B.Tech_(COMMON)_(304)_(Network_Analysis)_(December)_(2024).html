Network Analysis

Time : Three Hours Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जावेगा।

1.
a) Draw the parallel resonant circuit and derive the expression for resonant frequency.

समानांतर अनुनाद सर्किट बनाइए और अनुनाद आवृत्ति के लिए अभिव्यक्ति प्राप्त करें।

b) State and explain KCL and KVL with example.

उदाहरण के साथ KCL और KVL को बताएँ और समझाइए।

2.
a) Define Graph, oriented graph, Twigs and Cut set matrix.

ग्राफ, ओरिएंटेड ग्राफ, टहनियाँ और कट सेट मैट्रिक्स को परिभाषित करें।

b) What is incidence matrix? Find the complete incidence matrix for the graph shown in figure.

घटना मैट्रिक्स क्या है? चित्र में दिखाए गए ग्राफ के लिए संपूर्ण घटना मैट्रिक्स का पता लगाएं।

3.
a) Find the voltage drop across 3Ω resistor by superposition theorem.

सुपर पोजीशन प्रमेय द्वारा 3Ω रेसिस्टर के एक्रोस वोल्टेज ड्रॉप ज्ञात करें।

b) State and explain the Norton theorem with example.

उदाहरण के साथ नॉर्टन प्रमेय को बताइए एवं समझाइए।

4.
a) Find the Thevenin equivalent circuit for the following circuit.

निम्नलिखित सर्किट के लिए थेवेनिन समकक्ष सर्किट का पता लगाएं।

4.
b) State and explain maximum power transfer theorem for DC network.

DC नेटवर्क के लिये मैक्सिमम पावर ट्रांसफर प्रमेय लिखिए एवं समझाइये।

a) Find the value of i(0) using initial value theorem for the function I(s) = (2s + 3) / (s + 1)(s + 3).
