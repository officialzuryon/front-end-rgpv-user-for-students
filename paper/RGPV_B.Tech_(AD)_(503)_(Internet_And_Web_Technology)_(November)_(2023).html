Internet And Web Technology

Course: B.Tech Code: 503 Branch: AD Date: November 2023

Save as PDF

Opens your browser print dialog — select "Save as PDF" to download.

AD-503 (A) (GS)
[2]
B.Tech., V Semester
Examination, November 2023
Grading System (GS)
Internet and Web Technology
Time: Three Hours
Maximum Marks: 70

Note:

i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।
1. a)

How to handle HTTP requests and responses? Explain different kinds of HTTP request methods. 7

HTTP अनुरोधों और प्रतिक्रियाओं को कैसे संभालें? विभिन्न प्रकार की HTTP अनुरोध विधियों की व्याख्या करें।
b)

Describe the common problems and solutions in designing website in detail. 7

वेबसाइट डिजाइन करने में आने वाली सामान्य समस्याओं और समाधानों का विस्तार से वर्णन करें।
2. a)

Define HTML List tags. Write the HTML program that describes various types of lists with suitable example. 7

HTML सूची टैग को परिभाषित करें। वह HTML प्रोग्राम लिखें जो उपयुक्त उदाहरण के साथ विभिन्न प्रकार की सूचियों का वर्णन करता है।
b)

Differentiate HTML and XHTML. Write the HTML program that includes HTML frames and frame set. 7

HTML और XHTML में अंतर बताइए। वह HTML प्रोग्राम लिखें जिसमें HTML फ्रेम और फ्रेम सेट शामिल हों।
3. a)

Discuss how web socket connection is different from http connection in detail? 7

विस्तार से चर्चा करें कि वेब सॉकेट कनेक्शन http कनेक्शन से किस प्रकार भिन्न है?
b)

Write a JavaScript code for E-mail validation using HTML form. 7

HTML फॉर्म का उपयोग करके ई-मेल सत्यापन के लिए जावास्क्रिप्ट कोड लिखें।
4. a)

Distinguish between CSS2 and CSS3. Write the limitations of using entirely "pure" CSS. 7

CSS2 और CSS3 के बीच अंतर बताइए। पूरी तरह से "शुद्ध" CSS का उपयोग करने की सीमाएँ लिखें।
b)

Describe data binding in DHTML. 7

DHTML में डाटा बाइंडिंग का वर्णन करें।
5. a)

Write a simple javascript that performs arithmetic operations and display the result in an alert() dialog. 7

एक सरल जावास्क्रिप्ट लिखें जो अंकगणितीय संचालन करती है और परिणाम को अलर्ट() संवाद में प्रदर्शित करती है।
b)

Explain the different data types in XML schema. 7

XML स्कीमा में विभिन्न डाटा प्रकारों की व्याख्या करें।
6. a)

List and explain decision and looping concepts used in PHP. 7

PHP में प्रयुक्त निर्णय और लूपिंग अवधारणाओं की सूची बनाइए और समझाइए।
b)

Describe the various operators supported by PHP. 7

PHP द्वारा समर्थित विभिन्न ऑपरेटरों का वर्णन करें।
7. a)

Write a PHP script to delete data from an existing MySQL table. 7

मौजूदा MySQL तालिका से डाटा हटाने के लिए एक PHP स्क्रिप्ट लिखें।
b)

Write PHP code to create a login page for a web application and store details in database. 7

वेब एप्लिकेशन के लिए लॉगिन पेज बनाने और डेटाबेस में विवरण संग्रहीत करने के लिए PHP कोड लिखें।
8.

Write short notes on any Two: 14

किन्हीं दो पर संक्षिप्त टिप्पणी लिखिए।
  1. HTML5
  2. Session tracking and Cookies
  3. DTD
  4. HMTL Tags
