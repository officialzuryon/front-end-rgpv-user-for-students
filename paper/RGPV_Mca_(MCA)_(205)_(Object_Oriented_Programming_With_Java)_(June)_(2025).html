Roll No. .....................................
MCA-205
M.C.A. II Semester (Two Year Course)
Examination, June 2025
Object Oriented Programming with JAVA
Time: Three Hours
Maximum Marks: 70
Note: i) Attempt any five questions. किन्हीं पाँच प्रश्नों को हल कीजिए।
ii) All questions carry equal marks. सभी प्रश्नों के समान अंक हैं।
iii) In case of any doubt or dispute the English version question should be treated as final. किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जावेगा।
2.
(a)
What is the role of this pointer and the finalize() method in Java? How are they used with respect to memory allocation and garbage collection?
जावा में this पॉइंटर और finalize() विधि की क्या भूमिका है? मेमोरी आवंटन और कचरा संग्रहण के संबंध में इनका उपयोग कैसे किया जाता है?
(b)
Describe how to define and implement in interface in Java. How are variables in interfaces treated and how can interfaces be extended?
जावा में इंटरफेस को परिभाषित और कार्यान्वित करने का तरीका बताइए। इंटरफेस में चरों को किस तरह से व्यवहार किया जाता है और इंटरफेस को किस तरह से बढ़ाया जा सकता है?
3.
(a)
What are the life cycle methods of an applet in Java and how to HTML tags help in embedding an applet into a web page?
जावा में एप्लेट की जीवन चक्र विधियाँ क्या है और HTML टैग एप्लेट को वेब पेज में एम्बेड़ करने में कैसे मदद करते हैं?
(b)
Describe the role of try, catch, throw and finally in Java exception handling. How are constructors and finalizers used in the context of exception handling.
जावा अपवाद हैंडलिंग में try, catch, throw और finally की भूमिका का वर्णन करें। अपवाद हैंडलिंग के संदर्भ में कंस्ट्रक्टर और फाइनलाइजर का उपयोग कैसे किया जाता है?
4.
(a)
What is thread synchronization in Java and how does it help in solving the producer consumer problem? Illustrate with a simple code example.
जावा में थ्रेड सिंक्रनाइजेशन क्या है और यह उत्पादक-उपभोक्ता समस्या को हल करने में कैसे मदद करता है? एक सरल कोड उदाहरण के साथ समझाइए।
(b)
Explain the concept of inheritance in Java. What are its benefits and how does Java implement method overriding and dynamic method dispatch?
जावा में इनहेरिटेंस की अवधारणा को समझाइए। इसके क्या लाभ है और जावा मेथड ओवरराइडिंग और डायनेमिक मेथड डिस्पैच को कैसे लागू करता है?
(c)
Write a program to illustrate class and object in Java.
जावा में क्लास और ऑब्जेक्ट को चित्रित करने के लिए एक प्रोग्राम लिखें।
(b)
Explain the difference between creating a thread by extending the Thread class and implementing the Runnable interface. Which method is generally preferred and why?
थ्रेड क्लास को विस्तारित करके थ्रेड बनाने और रननेबल इंटरफेस को लागू करने के बीच अंतर को स्पष्ट करें। आमतौर पर कौन सी विधि को प्राथमिकता दी जाती है और क्यों?
5.
(a)
Differentiate between byte streams and character streams in Java. Describe the use of BufferedReader and BufferedWriter with examples.
जावा में बाइट स्ट्रीम और कैरेक्टर स्ट्रीम के बीच अंतर बताइए। उदाहरणों के साथ BufferedReader और BufferedWriter के उपयोग का वर्णन करें।
(b)
Explain the role of the JDBC-ODBC bridge and the DriverManager class in the JDBC connectivity model.
JDBC कनेक्टिविटी मॉडल में JDBC-ODBC ब्रिज और DriverManager क्लास की भूमिका की व्याख्या करें।
6.
(a)
Describe the class hierarchy of Java AWT window components. How does the Frame class fit into this hierarchy and what are its key features?
जावा AWT विंडो घटकों के वर्ग पदानुक्रम का वर्णन करें। फ्रेम वर्ग इस पदानुक्रम में कैसे फिट बैठता है और इसकी मुख्य विशेषताएँ क्या हैं?
(b)
Explain Java’s event delegation model. How does it improve the performance and structure of event-driven programs?
जावा के इवेंट डेलिगेशन मॉडल की व्याख्या करें। यह इवेंट-संचालित प्रोग्रामों के प्रदर्शन और संरचना को कैसे बेहतर बनाता है?
7.
(a)
What is the difference between GET and POST methods in servlets? How are request and response headers used in servlet communication?
सर्वलेट में GET और POST विधियों के बीच क्या अंतर है? सर्वलेट संचार में अनुरोध और प्रतिक्रिया हेडर का उपयोग कैसे किया जाता है?
(b)
Explain what are the basic elements of an HTML document? Explain the structure of an HTML page with an example.
HTML दस्तावेज के मूल तत्व क्या हैं? एक उदाहरण के साथ HTML पृष्ठ की संरचना को समझाइए।
8.
(a)
Explain the servlet life cycle with the help of the methods init(), service() and destroy().
init(), service() और destroy() विधियों की सहायता से सर्वलेट जीवन चक्र को समझाइए।
(b)
Explain the architecture of the Struts Framework.
स्ट्रट्स फ्रेमवर्क की वास्तुकला को समझाइए।
