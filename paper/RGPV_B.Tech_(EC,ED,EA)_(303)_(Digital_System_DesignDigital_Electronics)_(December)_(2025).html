EC/ED/EA 303 (GS)

B. Tech. III Semester

Examination, December 2025

Grading System (GS)

Digital System Design/Digital Electronics

Time : Three Hours Maximum Marks : 70

Note : (i) Attempt any five questions.

किन्ही पाँच प्रश्नों को हल कीजिए।

(ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

(iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.

(a) Convert the followings:

नीचे दिए गए को कन्वर्ट करें

(i) (753.4)₁₀ to decimal

(ii) (101101.101)₂ to hexadecimal

(iii) (9A.6)₁₆ to binary

(iv) (245)₁₀ to hexadecimal

(v) (1101101)₂ to octal

(vi) (59)₁₀ to Excess-3

(vii) (101011)₂ to Gray code

2.

(a) Minimize the function using K-map with don't care conditions:

डोंट केयर कंडीशन के साथ के-मैप का इस्तेमाल करके नीचे दिए गए फंक्शन को मिनिमाइज करें।

F=Σm(1, 3, 7, 11, 15)+d(0, 2, 5)

(b) Implement the following function using only NOR gates:

सिर्फ NOR गेट का इस्तेमाल करके नीचे दिया गया फंक्शन लागू करें।

F=(A+B)(C+D)

3.

(a) Explain the working of Carry Look Ahead Adder with logic equations.

लॉजिक इक्वेजन के साथ कैरी लुक अहेड एडर का काम समझाएँ।

(b) Design a 3-to-8 line decoder and explain its operation.

एक 3 से 8 लाइन डिकोडर डिजाइन करें और उसका ऑपरेशन समझाएँ।

4.

(a) Explain Master-Slave JK flip-flop with neat timing diagram. How does it eliminate the race-around condition?

मास्टर-स्लेव JK फ्लिप-फ्लॉप को साफ-सुथरे टाइमिंग डायग्राम के साथ समझाएं। यह रेस-अराउंड कंडीशन को कैसे खत्म करता है?

(b) Design a synchronous FSM using JK flip-flops for a mod-4 up counter. Draw state diagram and excitation table.

मॉड-4 अप काउंटर के लिए JK फ्लिप-फ्लॉप का इस्तेमाल करके सिंक्रोनस FSM का डिज़ाइन। स्टेट डायग्राम और एक्ससाइटेशन टेबल बनाएँ।

5.

(a) Design a Pseudo Random Binary Sequence (PRBS) generator and explain its working.

एक स्यूडो रैंडम बाइनरी सीक्वेंस (PRBS) जनरेटर डिजाइन कर और उसका काम समझाएँ।

(b) Explain asynchronous (ripple) counters with neat block diagram and timing waveforms. Discuss their advantages and limitations.

एसिंक्रोनस (रिप्पल) काउंटर को साफ-सुथरे ब्लॉक डायग्राम और टाइमिंग वेवफॉर्म के साथ समझाएं। उनके फायदे और लिमिटेशन पर चर्चा करें।

6.

(a) Explain the working of a universal shift register with neat diagram.

एक यूनिवर्सल शिफ्ट रजिस्टर के काम करने का तरीका साफ डायग्राम के साथ समझाएँ।

(b) Design a mod-6 synchronous counter using JK flip-flops by following these steps (i) State diagram (ii) State table (iii) Excitation table (iv) Logic diagram

इन स्टेप्स को फॉलो करके JK फ्लिप-फ्लॉप का इस्तेमाल करके एक mod-6 सिंक्रोनस काउंटर डिजाइन क��ें। (i) स्टेट डायग्राम (ii) स्टेट टेबल (iii) एक्ससाइटेशन टेबल (iv) लॉजिक डायग्राम

7.

(a) Explain Tri-state TTL logic with truth table and timing diagram. Why is tri-state logic required?

ट्रूथ टेबल और टाइमिंग डायग्राम के साथ ट्राई-स्टेट TTL लॉजिक समझाएँ। ट्राई-स्टेट लॉजिक क्यों जरूरी है?

(b) Implement the Boolean function F(A,B,C,D) = Σm(0,2,5,7,8,10,13,15) using a programmable logic device.

प्रोग्रामेबल लॉजिक डिवाइस का इस्तेमाल करके बूलियन फंक्शन F(A,B,C,D) = Σm(0,2,5,7,8,10,13,15) को इम्प्लीमेंट करें।

8.

Write a short note on any two:

इनमें से किन्ही दो पर एक छोटा नोट लिखें :

(a) De-Morgan's Theorem

डी-मॉर्गन थ्योरम

(b) Magnitude Comparator

मैग्नीट्यूड कम्पैरेटर

(c) Ring Counter

रिंग काउंटर

(d) Emitter Coupled Logic (ECL)

एमिटर कपलड लॉजिक
