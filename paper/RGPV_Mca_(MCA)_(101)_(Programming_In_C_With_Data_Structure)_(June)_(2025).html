M.C.A. I Semester (Two Year Course)

Examination, June 2025

Programming In C With Data Structure

Time : Three Hours

Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii)In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.

a) Explain the role of a preprocessor directive in C. How does the use of macros differ from functions?

C में प्रीप्रोसेसर डायेक्टिव्ह की भूमिका समझाइए। मेक्रेज का उपयोग फंक्शन्स से किस प्रकार भिन्न है ?

b) Describe the difference between automatic, static, register, and extern storage classes in C. How does the choice of storage class affect the lifetime and scope of variables?

C में स्वचालित, स्थिर, रजिस्टर और बाह्य स्टोरेज क्लासेस के बीच अंतर का वर्णन करें। भंडारण वर्ग का चयन वेरिएबल्स के जीवनकाल और दायरे को कैसे प्रभावित करता है?

2.

a) Write a C program that:

i) Creates a structure to store student information (roll number, name, marks in 3 subjects).

ii) Uses dynamic memory allocation to store information for n students.

iii) Calculates and displays the average marks for each student.

एक C प्रोग्राम लिखें जो:

अ) विद्यार्थी की जानकारी (रोल नंबर, नाम, 3 विषयों में अंक) संग्रहीत करने के लिए एक संरचना बनाता है।

ब) n विद्यार्थियों की जानकारी संग्रहीत करने के लिए गतिशील मेमोरी आवंटन का उपयोग करता है।

स) प्रत्येक छात्र के औसत अंकों की गणना और प्रदर्शन करता है।

3.

a) Define asymptotic notations. How do they help in analyzing algorithms?

एसिम्पटोटिक संकेतन को परिभाषित करें। वे एल्गोरिथम का विश्लेषण करने में कैसे मदद करते हैं?

b) Describe the contiguous array-based implementation of a stack. How would you handle stack overflow and underflow conditions?

स्टैक के सन्निहित सरणी-आधारित कार्यान्वयन का वर्णन करें। आप स्टैक ओवरफ्लो और अंडरफ्लो स्थितियों को कैसे संभालेंगे?

4.

Define a complete binary tree and an almost complete binary tree. How do they differ? Explain their importance in the context of heaps.

पूर्ण बाइनरी ट्री और लगभग पूर्ण बाइनरी ट्री को परिभाषित करें। वे कैसे भिन्न हैं? हीप के संदर्भ में उनके महत्व को समझाइए।

5.

a) How can a linked list be implemented using arrays?

श्रेणियों का उपयोग करके लिंक्ड सूची को कैसे क्रियान्वित किया जा सकता है?

b) What are the drawbacks of implementing a general list using contiguous memory?

सन्निहित मेमोरी का उपयोग करके सामान्य सूची को क्रियान्वित करने में क्या कमियाँ हैं?

6.

Write a C program to:

a) Implement a doubly linked list.

b) Allow the user to insert nodes at the beginning, at the end, and at a specific position.

c) Allow the user to delete a node from any specified position.

एक C प्रोग्राम लिखें जो:

अ) दोहरी लिंक्ड सूची का कार्यान्वयन करें।

ब) उपयोगकर्ता को आरंभ में, अंत में तथा विशिष्ट स्थान पर नोड्स डालने की अनुमति दें।

स) उपयोगकर्ता को किसी निर्दिष्ट स्थान से नोड हटाने की अनुमति दें।

7.

a) What is a threaded binary tree? How does threading help in efficient traversal? Provide a diagrammatic explanation.

थ्रेडेड बाइनरी ट्री क्या है? थ्रेडिंग कुशल ट्रैवर्सल में कैसे मदद करता है? एक आरेखीय स्पष्टीकरण प्रदान करें।

b) Differentiate between malloc() and calloc() functions in dynamic memory allocation. What are the possible dangers associated with improper memory deallocation?

डायनेमिक मेमोरी आवंटन में malloc() और calloc() फंक्शन के बीच अंतर बताइए। अनुचित मेमोरी डी-एलोकेशन से जुड़े संभावित खतरे क्या हैं?

8.

Write short notes on following:

i) Control Constructs and Iteration Constructs Functions

ii) Algorithm Complexity

iii) Conversion of forest into tree

निम्नलिखित पर संक्षिप्त टिप्पणियाँ लिखें।

i) नियंत्रण निर्माण और पुनरावृत्ति निर्माण कार्य

ii) एल्गोरिदम जटिलता

iii) फॉरेस्ट को ट्री में बदलना
