MCA-305 (3)
M.C.A. III Semester (Two Year Course)
Examination, December 2024
Distributed Systems
(Elective-III)
Time : Three Hours
Maximum Marks : 70
Note: i) Attempt any five questions.
किन्हीं पाँच प्रश्नों को हल कीजिए।
ii) All questions carry equal marks.
सभी प्रश्नों के समान अंक हैं।
iii) In case of any doubt or dispute the English version question should be treated as final.
किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।
1.
a)
List atleast seven differences between message oriented communication and stream oriented communication. 7
संदेश उन्मुख संचार और स्ट्रीम उन्मुख संचार के बीच कम से कम सात अंतरों की सूची बनाइये।
b)
What are the Client Stubs and Server Stubs? How are they used in Remote Procedure Calls (RPC)? 7
क्लाइंट स्टब्स और सर्वर स्टब्स क्या हैं? रिमोट प्रोसीजर कॉल (RPC) में इनका उपयोग कैसे किया जाता है?
2.
a)
Define the thread and process. Would it make sense to limit the number of threads in a server processes? 7
थ्रेड और प्रोसेस को परिभाषित करें। क्या सर्वर प्रोसेस में थ्रेड की संख्या को सीमित करना उचित होगा?
b)
Explain bully algorithm and ring algorithm with suitable example. 7
उपयुक्त उदाहरण के साथ बुली एल्गोरिथम और रिंग एल्गोरिथम को समझाइए।
3.
a)
The Diffie-Hellman key exchange protocol can also be used to establish a shared secret key between three parties. Explain how? 7
डिफ्फी-हेलमैन कुंजी विनिमय प्रोटोकॉल का उपयोग तीन पक्षों के बीच साझा गुप्त कुंजी स्थापित करने के लिए भी किया जा सकता है। समझाइए कैसे?
b)
Compare Data Centric Consistency Model with Client Centric Consistency Model with suitable example. 7
उपयुक्त उदाहरण के साथ डेटा सेंट्रिक कंसिस्टेंसी मॉडल की क्लाइंट सेंट्रिक कंसिस्टेंसी मॉडल के साथ तुलना करें।
4.
a)
Describe the object model of CORBA. Also discuss the services provided by CORBA system. 7
CORBA के ऑब्जेक्ट मॉडल का वर्णन करें। CORBA सिस्टम द्वारा प्रदान की जाने वाली सेवाओं पर भी चर्चा करें।
b)
Discuss the file system model, processes and naming in Sun Network File System. 7
सन नेटवर्क फाइल सिस्टम में फाइल सिस्टम मॉडल, प्रक्रियाओं और नामकरण पर चर्चा करें।

5.
a)
Summarize the advantages and disadvantages of a distributed shared memory system. Also, explain the principal operations of a page-based DSM system. 7
वितरित साझा मेमोरी सिस्टम के फायदे और नुकसान का संक्षेप में वर्णन करें। इसके अलावा, पेज-आधारित DSM प्रणाली के प्रमुख संचालन की व्याख्या करें।
b)
Write minimum seven design issues of distributed file system with correct explanation. 7
वितरित फाइल सिस्टम के कम से कम सात डिजाइन मुद्दों को सही स्पष्टीकरण के साथ लिखें।
6.
a)
What is object replication? Explain atleast four reasons of data replication. 7
ऑब्जेक्ट प्रतिकृति क्या है? डेटा प्रतिकृति के कम से कम चार कारण बताइए।
b)
Explain the notion of coordination in distributed systems and present an overview of JINI. 7
वितरित प्रणालियों में समन्वय की अवधारणा को समझाइए तथा JINI का अवलोकन प्रस्तुत कीजिए।
7.
a)
External synchronization ensures internal synchronization. But the vice versa does not stand true. Justify. 7
बाह्य समन्वय आंतरिक समन्वय सुनिश्चित करता है। लेकिन इसके विपरीत सत्य नहीं है। औचित्य सिद्ध करें।
b)
Why do we need a distributed system? List out the types of distributed file system. 7
हमें वितरित सिस्टम की आवश्यकता क्यों है? वितरित फाइल सिस्टम के प्रकारों की सूची बनाइए।
8.
Write short note on any four of the followings: 14

c)
