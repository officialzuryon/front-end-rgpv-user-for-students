Internet And Web Technology

Course: B.Tech Code: 503 Branch: AD Date: November 2022

AD-503 (A) (GS)

B.Tech., V Semester

Examination, November 2022

Grading System (GS)

Internet and Web Technology

Time: Three Hours

Maximum Marks: 70

Note:

  1. i) Answer any five questions.
  2. ii) All questions carry equal marks.
  3. iii) In case of any doubt or dispute the English version question should be treated as final.

नोट:

  1. i) किन्हीं पाँच प्रश्नों को हल कीजिए।
  2. ii) सभी प्रश्नों के समान अंक हैं।
  3. iii) किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।
1.

a) Explain Web design issues including Browser.

ब्राउज़र सहित वेब डिजाइन के मुद्दों की व्याख्या करें ।

7

b) What is HTTP? Explain its utility and the various methods used by HTTP.

HTTP क्या है? इसकी उपयोगिता और HTTP द्वारा उपयोग की जाने वाली विभिन्न विधियों की व्याख्या कीजिए ।

7
2.

a) Write a Java Script which displays current date and time.

एक जावा स्क्रिप्ट लिखें जो वर्तमान दिनांक और समय प्रदर्शित करती है ।

7
3.

a) Discuss the difference between HTML and XHTML with respect to elements.

तत्वों के संबंध में HTML और XHTML के बीच अंतर की चर्चा करें

7

b) Explain in detail DOM event handling. Also explain with an example of creating a context menu.

DOM इवेंट हैंडलिंग को विस्तार से समझाइए। संदर्भ मेनू बनाने के उदाहरण के साथ भी समझाइए।

7
4.

a) What is XML? What are the attributes of XML?

XML क्या है? XML की विशेषताएँ क्या हैं?

7

b) Explain the concept of object oriented programming with PHP.

PHP के साथ ऑब्जेक्ट ओरिएंटेड प्रोग्रामिंग की अवधारणा की व्याख्या करें।

7
5.

a) Write details using with PHP database creating, selecting, listing.

PHP डाटाबेस बनाने, चयन करने, लिस्टिंग के साथ विवरण लिखें ।

7

b) Write short note on radio buttons and check boxes object in Java Script.

जावा स्क्रिप्ट में रेडियो बटन और चेक बॉक्स ऑब्जेक्ट पर संक्षिप्त नोट लिखें।

7
6.

a) What are FORMS? Explain different FORM fields.

फॉर्म क्या होते है? विभिन्न प्रपत्र क्षेत्रों की व्याख्या करें ।

7

b) Write an HTML document to provide a form that collect hammer and phone numbers.

एक प्रपत्र प्रदान करने के लिए एक HTML दस्तावेज लिखें जो हथौड़ा और फोन नंबर एकत्र करता है ।

7
7.

a) Write a PHP script which takes the user name, password and e-mail values from user and checks whether the user has filled the textboxes or not. Also, check the e-mail field whether it is incorrect format or not.

एक PHP स्क्रिप्ट लिखें जो उपयोगकर्त�� से उपयोगकर्ता नाम, पासवर्ड और ई-मेल मान लेता है और जाँचता है कि उपयोगकर्ता ने टेक्स्टबॉक्स भरे हैं या नहीं। साथ ही, ई-मेल फ़ील्ड की जाँच करें कि यह गलत प्रारूप है या नहीं।

7

b) Write any five methods of Math object of java script.

जावा लिपि की गणित वस्तु की कोई पाँच विधियाँ लिखिए।

7
8.

a) Write short notes on:

  1. Web browser
  2. Identifier in java script
  3. Webhosting

संक्षिप्त नोट्स लिखें।

  1. वेब ब्राउजर
  2. जावा स्क्रिप्ट में पहचानकर्ता
  3. वेब होस्टिंग
7

b) Write down the advantages and disadvantages of using style sheets.

स्टाइल शीट का उपयोग करने के फायदे और नुकसान लिखिए।
