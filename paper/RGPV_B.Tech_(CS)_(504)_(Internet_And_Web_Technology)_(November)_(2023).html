Internet And Web Technology

Course: B.Tech Code: 504 Branch: CS Date: November 2023

Roll No .......................................

CS-504 (A) (GS)

B.Tech., V Semester

Examination, November 2023

Grading System (GS)

Internet and Web Technology

Time : Three Hours

Maximum Marks : 70

Note: i) Attempt any five questions.

किन्हीं पाँच प्रश्नों को हल कीजिए।

ii) All questions carry equal marks.

सभी प्रश्नों के समान अंक हैं।

iii) In case of any doubt or dispute the English version question should be treated as final.

किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.
a)

Explain the various types of web design issues.

विभिन्न प्रकार के वेब डिजाइन मुद्दों की व्याख्या करें।

b)

What are the steps involved in planning and publishing a website? Explain.

किसी वेबसाइट की योजना बनाने और उसे प्रकाशित करने में क्या चरण शामिल होते हैं? वर्णन करें।

2.
a)

Describe the impact of Web 2.0 on web development.

वेब विकास पर वेब 2.0 के प्रभाव का वर्णन करें।

b)

Differentiate HTML and XHTML with suitable examples.

उपयुक्त उदाहरणों के साथ HTML और XHTML में अंतर बताएं।

3.
a)

Write an HTML program to divide your browsing window into 3 parts consisting of 2 rows of size 20% and 80% and divide the last row into two columns of size 30% and 70%.

अपनी ब्राउज़िंग विंडो को 3 भागों में विभाजित करने के लिए HTML प्रोग्राम लिखें, जिसमें 20% और 80% आकार की 2 पंक्तियाँ हों और अंतिम पंक्ति को 30% और 70% आकार के दो कॉलम में विभाजित करें।

b)

Explain the properties and methods of objects in JavaScript with a suitable script.

एक उपयुक्त स्क्रिप्ट के साथ जावास्क्रिप्ट में वस्तुओं के गुणों और विधियों को समझाइए।

4.
a)

Explain how to communicate DOM API?

बताइए कि DOM API को कैसे संचारित किया जाए?

b)

How will CSS improve the web page and website design layout?

CSS वेब पेज और वेबसाइट डिज़ाइन लेआउट को कैसे सुधारेगा?

5.
a)

Compare the critical differences between DTD and XML Schema.

DTD और XML स्कीमा के बीच महत्वपूर्ण अंतरों की तुलना करें।

b)

Briefly explain XSL transformation with an example of script.

एक उदाहरण स्क्रिप्ट के साथ XSL परिवर्तन को संक्षेप में समझाइए।

6.
a)

Explain form handling with the get and post methods in PHP.

PHP में प्राप्त और पोस्ट विधियों के साथ फ़ॉर्म प्रबंधन को समझाइए।

b)

What is the difference between sessions and cookies in PHP?

PHP में सत्र और कुकीज़ के बीच क्या अंतर है?

7.
a)

Explain the steps in the PHP code for querying a database with suitable examples.

उपयुक्त उदाहरणों के साथ डेटाबेस को क्वेरी करने के लिए PHP कोड के चरणों की व्याख्या करें।

b)

Create a simple shopping cart application using PHP and MySQL.

PHP और MySQL का उपयोग करके एक सरल शॉपिंग कार्ट एप्लिकेशन बनाइए।

8.

Write a short note on any two:

किन्हीं दो पर एक संक्षिप्त टिप्पणी लिखें।

a)

PHPmyadmin and database bugs

PHPmyadmin और डेटाबेस बग

b)

Client-side scripting and Server-side scripting

क्लाइंट-साइड स्क्रिप्टिंग और सर्वर-साइड स्क्रिप्टिंग

c)

Features of HTML5

HTML5 की विशेषताएँ

d)

HTTP Protocol

HTTP प्रोटोकॉल
