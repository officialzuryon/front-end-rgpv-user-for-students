किसी भी प्रकार के संदेह अथव�� विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।

1.

a) Using Newton-Raphson method find the real root of the equation 3x = cosx + 1 correct to five decimal places.

न्यूटन-राफ्सन विधि का उपयोग करके समीकरण 3x = cosx + 1 का वास्तविक मूल पाँच दशमलव स्थानों तक ज्ञात करें।

b) Find f(g) from the following table

निम्नलिखित तालिका से f(g) ज्ञात कीजिए।

x 0 1 4 5 7
f(x) 8 11 68 123 163

2.

a) Apply the Simpson's 3/8 rule to evaluate the following integral for six digit ∫₀¹ x^(1/3) dx.

छः अंकीय ∫₀¹ x^(1/3) dx के लिए निम्नलिखित समाकल का मूल्यांकन करने के लिए सिम्पसन के 3/8 नियम को लागू करें।

b) Solve the following system of equation using Gauss-Seidel method.

गॉस-साइडल विधि का उपयोग करके निम्नलिखित समीकरण प्रणाली को हल करें।

27x + 6y - z = 85

6x + 15y + 2z = 72

x + y + 54z = 110

3.

a) Using Runge-Kutta method of fourth order solve dy/dx = (y^2 - x^2) / (y^2 + x^2), y(0) = 1 at x = 0.2 in step of 0.2.

चौथे क्रम की रंगे-कुट्टा विधि का उपयोग करके x = 0.2 पर dy/dx = (y^2 - x^2) / (y^2 + x^2), y(0) = 1 को 0.2 के चरण में हल करें।

b) Find y(2.2) using Euler's method for dy/dx = -xy^2 where y(2) = 1.

dy/dx = -xy^2 के लिए यूलर विधि का उपयोग करके y(2.2) ज्ञात करें जहाँ y(2) = 1 है।

4.

a) Find the Inverse Laplace transform of 1 / (P^2 (P^2 + a^2)).

1 / (P^2 (P^2 + a^2)) का व्युत्क्रम लाप्लास रूपांतर ज्ञात कीजिए।

मुख्य लाप्लास रूपांतरण निम्न है:

∫₀^∞ e^(-st) f(t) dt

ज्ञात करें:

∫₀^∞ e^(-at) sin(bt) dt / t

इसका रूपांतरण है:

tan^(-1) (b/s) - tan^(-1) (b/a)

जहां a=0 और b=1,

tan^(-1) (1/s) - tan^(-1) (1/0)

= tan^(-1) (1/s) - π/2

= -cot^(-1) (1/s)

= -tan^(-1) (s)

b) Using Laplace transform solve the differential equation d^2y/dt^2 + 2dy/dt + 2y = 0.

लाप्लास रूपांतरण का उपयोग करके अंतर समीकरण d^2y/dt^2 + 2dy/dt + 2y = 0 हल करें।

Where y = 1, dy/dt = 2 at t = 0.

हल करें जहाँ y = 1, dy/dt = 2 पर t = 0।

5.

a) Fit a Poisson distribution to the following.

निम्नलिखित में पॉइसन वितरण फिट करें।

x 0 1 2 3 4
f(x) 46 38 22 9 1

b) Calculate the mean and standard deviation of Binomial Distribution.

द्विपद वितरण के माध्य और मानक विचलन की गणना करें।

6.

a) Find dy/dx at x = 1.5 from the following table.

निम्न तालिका से x = 1.5 पर dy/dx ज्ञात करें।

x 1.5 2.0 2.5 3.0 3.5 4.0
f(x) 3.375 7.0 13.625 24.0 38.87 59.0

7.

a) Show that Newton Raphson method is quadratic convergent.

दर्शाइये कि न्यूटन-राफ्सन विधि द्विघाती अभिसारी है।

b) Drive Newton forward interpolation formula and use it to estimate the value of f(1.25) from the following table.

न्यूटन फॉरवर्ड इंटरपोलेशन फॉर्मूला का उपयोग करें और निम्न तालिका से f(1.25) के मान का अनुमान लगाने के लिए इसका उपयोग करें।

x 1.0 1.5 2.0 2.5
f(x) 4.00 18.25 44.00 84.25

3x + y - z = 3

Using Gauss elimination method.

समीकरणों की प्रणाली को हल करें।

2x - 8y + z = -5

x - 2y + 9z = 8

1/x का फूरियर साइन रूपांतर ज्ञात करें।
