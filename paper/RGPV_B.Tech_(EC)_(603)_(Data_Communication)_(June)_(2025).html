Data Communication

Course: B.Tech Code: 603 Branch: EC Date: June 2025

Total No of Questions: 8 Total No of Printed Pages: 4
Roll No

EC-603 (A) (GS)

B.Tech., VI Semester

Examination, June 2025

Grading System (GS)

Data Communication

Time: Three Hours Maximum Marks: 70
Note:
i) Attempt any five questions.
किन्हीं पाँच प्रश्नों को हल कीजिए।
ii) All questions carry equal marks.
सभी प्रश्नों के समान अंक हैं।
iii) In case of any doubt or dispute the English version question should be treated as final.
किसी भी प्रकार के संदेह अथवा विवाद की स्थिति में अंग्रेजी भाषा के प्रश्न को अंतिम माना जायेगा।
1.
a) What is Data communication? List and briefly describe the five main components of a data communication system.
डाटा संचार क्या है? डाटा संचार प्रणाली के पाँच मुख्य घटकों की सूची बनाइए और उनका संक्षेप में वर्णन करें।
b) Calculate the time required to transmit 1024 bytes of data using serial transmission at 9600 bps. Compare this with the time required for parallel transmission assuming an 8-bit parallel channel with a clock rate of 1 MHz.
9600 bps पर सीरियल ट्रांसमिशन का उपयोग करके 1024 बाइट्स डाटा संचालित करने के लिए आवश्यक समय की गणना करें। 1 मेगाहर्ट्ज की क्लॉक दर के साथ 8-बिट समानांतर चैनल मानकर समानांतर ट्रांसमिशन के लिए आवश्यक समय के साथ इसकी तुलना करें।
2.
a) Explain the concept of network topology. Briefly describe the following topologies: bus, star, ring and mesh.
नेटवर्क टोपोलॉजी की अवधारणा की व्याख्या करें। निम्नलिखित टोपोलॉजी का संक्षेप में वर्णन करें: बस, स्टार, रिंग और मेश
b) Draw the seven layers architecture of OSI model and describe the main function of each layer.
OSI मॉडल की सात परतों की वास्तुकला बनाइए और प्रत्येक परत के मुख्य कार्य का वर्णन करें।
3.
a) Describe the various types of transmission impairments that can affect communication signals and what measures can be taken to mitigate these impairments.
संचार संकेतों को प्रभावित करने वाली विभिन्न प्रकार की संचरण कमियों का वर्णन करें और इन कमियों को कम करने के लिए क्या उपाय किए जा सकते हैं?
b) What is Message switching? How does it differ from packet switching? Discuss the advantages and disadvantages of message switching.
संदेश स्विचिंग क्या है? यह पैकेट स्विचिंग से किस प्रकार भिन्न है? संदेश स्विचिंग के लाभ और हानि पर चर्चा करें।
4.
a) Describe the different methods of framing in data communication. How do these methods help in error detection and data integrity.
डाटा संचार में फ्रेमिंग के विभिन्न तरीकों का वर्णन करें। ये तरीके त्रुटि पहचान और डाटा अखंडता में किस प्रकार मदद करते हैं?
b) A message "1011001" needs to be transmitted using a Longitudinal Redundancy Check (LRC). Show how the LRC code is generated and how it helps in error detection.
संदेश "1011001" को अनुदैर्ध्य अतिरेक जाँच (LRC) का उपयोग करके प्रेषित किया जाना चाहिए। दिखाएँ कि LRC कोड कैसे उत्पन्न होता है और यह त्रुटि पहचान में किस प्रकार मदद करता है?
5.
a) Explain the High-Level Data Link Control (HDLC) protocol. What are its main features and modes of operation?
हाई-लेवल डाटा लिंक कंट्रोल (HDLC) प्रोटोकॉल की व्याख्या करें। इसकी मुख्य विशेषताएँ और संचालन के तरीके क्या हैं?
b) Compare and contrast Token Bus and Token Ring LAN architectures.
टोकेन बस और टोकेन रिंग LAN आर्किटेक्चर की तुलना और अंतर बताइए।
6.
a) Discuss the architecture and operation of the Distributed Queue Dual Bus (DQDB) protocol specified in the IEEE 802.6 standard. How does it support data transmission in a MAN?
IEEE 802.6 मानक में निर्दिष्ट वितरित कतार दोहरी बस (DQDB) प्रोटोकॉल की वास्तुकला और संचालन पर चर्चा करें। यह MAN में डाटा ट्रांसमिशन का समर्थन कैसे करता है?
b) What is Asynchronous Transfer Mode (ATM)? Describe its cell-based architecture and switching mechanisms.
एसिंक्रोनस ट्रांसफर मोड (ATM) क्या है? इसकी सेल-आधारित वास्तुकला और स्विचिंग तंत्र का वर्णन करें।
7.
a) Discuss the roles and functions of repeaters, bridges, gateways and other networking devices in a computer network.
कम्प्यूटर नेटवर्क में रिपीटर्स, ब्रिज, गेटवे और अन्य नेटवर्किंग डिवाइसेज की भूमिका और कार्यों पर चर्चा करें।
b) Define link state routing. How does it differ from distance vector routing in terms of routing table updates and route calculation?
लिंक स्टेट रूटिंग को परिभाषित करें। रूटिंग टेबल अपडेट और रूट गणना के संदर्भ में यह डिस्टेंस वेक्टर रूटिंग से किस प्रकार भिन्न है?
8.
Write a short note on any two:
किन्हीं दो पर संक्षिप्त टिप्पणी लिखें।
a) Line Encoding
अ) लाइन एनकोडिंग
b) Stop and Wait Protocol
ब) स्टॉप एंड वेट प्रोटोकॉल
c) FDDI
स) FDDI
d) TCP/IP
द) TCP/IP का अवलोकन
