1. a)

Explain about Page Layout and linking in brief.

पेज लेआउट और लिंकिंग के बारे में संक्षेप में बताइए।

b)

What is HTTP? Explain its utility and the various methods used by HTTP.

HTTP क्या है? इसकी उपयोगिता और HTTP द्वारा उपयोग की जाने वाली विभिन्न विधियों की व्याख्या कीजिए।

2. a)

Explain various types of lists that can be created in HTML by giving suitable example.

HTML में बनाई जा सकने वाली विभिन्न प्रकार की सूचियों को उपयुक्त उदाहरण देते हुए समझाइए।

b)

Discuss the difference between HTML and XHTML with respect to elements.

तत्वों के संबंध में HTML और XHTML के बीच अंतर चर्चा करें।

Write short note on radio buttons and check boxes object in Java Script.

जावास्क्रिप्ट में रेडियो बटन और चेक बॉक्स ऑब्जेक्ट पर संक्षिप्त नोट लिखें।

b)

Explain about CSS and its advantages in brief.

CSS और इसके फायदों के बारे में संक्षेप में बताइए।

4. a)

XML क्या है? XML की विशेषताएँ क्या हैं?

b)

Explain about Loop Statements in PHP.

PHP में लूप स्टेटमेंट के बारे में समझाइए।

5. a)

Explain the concept of object oriented programming with PHP.

PHP के साथ ऑब्जेक्ट ओरिएंटेड प्रोग्रामिंग की अवधारणा को समझाइए।

b)

What is File? Explain how file is created in PHP?

फाइल क्या है? समझाइए कि PHP में फाइल कैसे बनाई जाती है?

Write the connectivity string in PHP with MySQL database.

Write a PHP program for creating a table.
